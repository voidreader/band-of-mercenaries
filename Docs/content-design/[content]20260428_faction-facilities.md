## 개요

14개 세력별로 가입 시 건설 가능한 **고유 시설 1종**을 설계한다. 본 시설은 가입 세력의 **장기 충성 보상**이자 **세력 색깔 강화** 장치다. 기존 12종 시설(보편 인프라) + 14개 세력 패시브(상시 보너스) 위에 얹히는 **세 번째 보너스 풀**로, 세력 정체성에 직결된 새 메커닉을 도입한다.

**M4 시설 설계 3원칙:**

1. **독립 효과형**: 14종 모두 기존 시설/패시브와 차원이 다른 새 보너스를 제공한다. 기존 시설 효과 강화는 패시브가 이미 담당하므로 시설은 중복하지 않는다.
2. **세력 색깔 강화**: 세력의 패시브가 정의한 정체성을 한 발 더 깊게 표현한다 (예: 전사 길드 패시브 = raid/hunt 성공률 +5% → 전용 시설 = 대성공 확률 가산).
3. **트레잇 해금 분리**: 트레잇 해금은 페이즈 1 #4 "세력 전용 트레잇 컨셉"에서 평판 + 거점 방문 + 행동 지표 게이트로 처리하므로, 시설은 메커닉 보정만 담당한다.

## 레퍼런스 분석

| 레퍼런스 | 차용 메커니즘 | 적용 방식 |
|---------|-------------|----------|
| OGame — 종족별 보너스 건물 | 종족 선택이 특정 건물 트리를 해금 | 가입 세력 1개당 전용 시설 1개 해금, 탈퇴 시 잠금 |
| Path of Exile — Ascendancy Passives | 직업 선택 후 ascendancy로 전문화 깊이 추가 | 패시브(가입 즉시) + 시설(평판/골드 투자)로 두 단계 전문화 |
| Stellaris — Ascension Perks | 제국 정체성에 따라 고유 perk 트리 해금 | 14세력 시설이 14개 perk 트리 역할, 다중 가입(최대 3)으로 조합 가능 |
| Caves of Qud — Mutation Trees | 단일 변이 카테고리 내 깊이 vs 다중 카테고리 폭 트레이드오프 | 한 세력 시설 풀레벨(Lv15) vs 다세력 시설 분산(각 Lv5) 선택 |

**현 게임과의 차별 포인트**: 기존 12종 시설은 "용병단 인프라 = 누구나 짓는 보편 시설"이고, 패시브는 "가입 즉시 적용되는 정적 보너스"다. M4 시설은 **"가입 + 골드 + 시간 투자가 결합된 충성 보상"**이라는 새 축을 만든다. 이는 평판 91~100(핵심) 도달과 함께 풀레벨이 가능한 "세력 끝물 컨텐츠"의 골드 싱크가 된다.

## 상세 설계

### 1. 공통 규칙

#### 1.1 비용 곡선 (4티어 분류 적용)

기존 시설 4티어 분류(`balance-design/20260414_facility_balance.md`)를 그대로 차용하되, 세력 시설은 **효과 영향도**로 분류한다:

| 카테고리 | 기준 | 세력 시설 |
|---------|------|---------|
| **Expensive (1,000G)** | 직접 골드/대성공/장비 영향 (5종) | 교역소 / 무기고 / 금서 열람실 / 개조 실험실 / 룬 대장간 |
| **Premium (700G)** | 정성 효과(트레잇/조사/모집) 영향 (7종) | 의뢰 중개소 / 마법 연구실 / 태양 신전 / 감시 초소 / 자연 제단 / 혈계 도서관 / 부족 훈련장 |
| **Standard (500G)** | 부수 비용 절감 효과 (2종) | 암시장 / 용암 단련장 |

**비용 공식**: 기존과 동일.
- Lv1 비용 = base × 0.1 (Expensive 100G / Premium 70G / Standard 50G)
- Lv2 비용 = base × 0.2 (Expensive 200G / Premium 140G / Standard 100G)
- Lv3+ 비용 = base × 1.5^(Lv-3) (cost_multiplier=1.5 동일)

#### 1.2 최대 레벨

**전 시설 일률 Lv15.** 이유:

- 평판 91~100(핵심) 구간 도달 ≈ 풀레벨 도달 시점이 되도록 설계 (체감 동기화)
- 14세력 동시 가입은 불가능하므로 한 세력당 풀 투자 시 ~270K~390K G 소요 (Premium~Expensive 기준)
- 시설 탭 UI 부담 완화: 가입 1세력 기준 12 + 1 = 13개 시설만 동시 노출

> **대안 검토**: Lv25 차등 → 풀 투자 비용 ~3M~10M G로 비현실적. 기각.
> **대안 검토**: Lv10 균일 → 효과 곡선이 너무 빨리 포화. 기각.

#### 1.3 효과 곡선 공식

기존 시설은 `max_effect × ln(1 + Lv × α) / ln(1 + 25 × α)`(Lv25 기준 정규화)다. 세력 시설은 Lv15 일률이므로:

```
세력 시설 효과(Lv) = max_effect × ln(1 + Lv × α) / ln(1 + 15 × α)
```

→ **Lv15에서 정확히 max_effect 도달**. α 파라미터는 기존 시설과 동일 규약:
- α=0.3 (비율 효과): 트레잇/모집/성공률/회복 보정 등
- α=0.4 (카운트 효과): 발견 슬롯, 사용 횟수 등

> **세부 공식 확정은 페이즈 2(`/balance-designer`)에 위임.** 본 문서는 max_effect 값과 α 분류만 제시.

#### 1.4 건설 시간

기존 시설과 동일 공식:
- Lv1: 5분 / Lv2: 10분 / Lv3+: 25 × 1.45^(Lv-3) 분
- Lv15까지 누적 ~17일

**건설 큐 공유 vs 분리**: 본 문서에서는 **공유 큐 권장**. 이유:
- 가입 세력은 보통 1~3개로 한정되므로 별도 큐를 만들 정도의 동시성 압박이 없음
- 기존 `UserData.constructionFacilityId` Hive 필드 그대로 사용 → 마이그레이션 불필요
- 플레이어 의사결정에 "12개 + 세력 N개 시설 중 무엇을 우선할 것인가"라는 자연스러운 트레이드오프 추가
- (페이즈 4 명세 단계에서 최종 결정. 별도 큐로 분리해야 한다는 강한 근거 발견 시 변경 가능)

#### 1.5 탈퇴 시 잠금/레벨 보존 정책

| 상태 | 시설 가용 | 효과 적용 | 건설/업그레이드 | 비고 |
|------|----------|----------|---------------|------|
| **가입 중 (평판 > 0)** | ✅ 활성 | ✅ 적용 | ✅ 가능 | 정상 상태 |
| **자발 탈퇴** | 🔒 잠금 | ❌ 미적용 | ❌ 차단 | 레벨 보존 |
| **충돌 강제 탈퇴** | 🔒 잠금 | ❌ 미적용 | ❌ 차단 | 레벨 보존 + 평판 -20 |
| **적대 (평판 ≤ 0)** | 🔒 잠금 | ❌ 미적용 | ❌ 차단 | 자발 탈퇴와 동일 처리 |
| **재가입** | ✅ 활성 | ✅ 즉시 재적용 | ✅ 보존 레벨에서 이어서 | 골드/시간 재투자 불필요 |

**핵심 규칙:**

1. **레벨은 영구 보존** — `FactionState.facilityLevels` (HiveField(5), 이미 존재) 활용. 잠금 상태에서도 데이터는 유지.
2. **잠금 중에는 효과 0** — `PassiveBonusService` 조회 시 가입 상태(`isJoined`)를 먼저 확인하고 미가입이면 시설 효과를 합산하지 않음.
3. **재가입 페널티 없음** — 충돌 강제 탈퇴 후 재가입 시 평판 -20 페널티는 적용되지만 시설 레벨은 그대로. 시설 자체에는 추가 손실 없음.
4. **건설 진행 중 탈퇴** — 진행 중 건설은 즉시 취소되며, **소비된 골드/시간 환불 없음** (의도된 매몰 비용). 부분 진척은 폐기되고 잠금된 현재 레벨이 유지.
5. **다른 세력에 시설 이전 불가** — 시설은 세력별로 1:1 고정. 모험가 길드의 의뢰 중개소 Lv10이 도둑 길드의 암시장 Lv0을 채우지 않음.

#### 1.6 다중 가입 시 다중 시설 가동

가입 세력이 2~3개일 때, **각 세력 시설은 독립적으로 활성화**된다. 패시브 스태킹 규칙(`[content]20260417_faction_passive_mapping.md` 섹션 3)을 그대로 차용:

- 같은 효과 타입은 가산/곱셈 규칙에 따라 합산
- 다른 효과 타입은 독립 적용

특정 시설 효과가 다른 세력 시설 효과와 직접 충돌할 가능성은 낮다 (각 시설이 새 메커닉 영역을 점유하므로). 단, 일부 효과(예: 의뢰 중개소 + 게시판 = "고품질 퀘스트 출현 확률 ↑")는 기존 시설 또는 다른 세력 시설과 자연스럽게 합산되며, 이는 **의도된 시너지**다.

### 2. 14종 시설 카탈로그

각 시설은 다음 6개 항목을 갖는다:
- **이름** (한글)
- **설명** (세력 정체성 반영)
- **효과 방향성** (메커닉 영역 + max_effect 후보값)
- **카테고리** (Core/Standard/Premium/Expensive)
- **기존 시설 12종과의 시너지/중복 검토**
- **세력 패시브와의 차별화**

> 효과 타입의 세부 JSON 스키마는 페이즈 2 밸런스 검토 후 페이즈 4 명세에서 확정.

---

#### 공개 세력 (6종)

##### 1. 모험가 길드 — **의뢰 중개소** (Premium, α=0.3)

> 대륙 전역의 모험가 지부와 직통으로 연결된 의뢰 게시 사무소. 길드 본부에서 고급 의뢰가 직접 흘러들어온다.

- **효과**: 현재 리전의 퀘스트 풀에 **explore/raid 계열 고품질 의뢰**가 추가 노출될 확률 가산. 신규 효과 타입 `quest_pool_quality_bonus` 도입 후보.
  - 운영: 퀘스트 자동 생성 시 quest_pool 가중치 계산 단계에서 reward_multiplier가 평균 이상인 풀의 가중치를 ×(1+value)로 부스트.
  - max_effect 후보: 0.40 (Lv15에서 +40% 가중치 부스트)
- **카테고리 근거**: 골드 보상 우상향 효과 (직접 골드 영향). 단, 보장된 보상이 아닌 확률적이라 Premium 분류.
- **기존 시설 시너지/중복**:
  - 정보망(퀘스트 슬롯 +5개) → **보완**. 슬롯 수와 품질은 독립 차원.
  - 게시판(고보상 확률 +35%, 현재 stub) → **합산 시너지 허용**. 게시판이 "전체 고보상 확률"이라면 의뢰 중개소는 "explore/raid 한정 고품질 풀 가중치". quest_type별로 서로 다른 풀에 영향이라 중복도 미미.
- **패시브 차별화**: 모험가 길드 패시브 = `quest_reward_multiplier` (탐험 보상 +12%, 결과값 배율). 시설 = 풀 가중치 부스트 → 그 결과 평균 보상이 우상향. 차원이 다름 (정적 배율 vs 풀 분포 변경).

---

##### 2. 상인 연합 — **교역소** (Expensive, α=0.3)

> 대륙 교역망의 환전 거점. 임무 완수와 함께 들고 온 전리품을 즉시 골드로 환전하며, 길드 차원의 환율 우대를 받는다.

- **효과**: 모든 퀘스트 완료 시 **추가 골드 환전 보너스** (보상 골드 ×(1+value)). 신규 효과 타입 `quest_reward_gold_bonus_facility` (패시브와 별도 풀).
  - max_effect 후보: 0.20 (Lv15에서 +20% 추가 골드)
- **카테고리 근거**: 직접 골드 우상향 (모든 퀘스트 영향). Expensive 합당.
- **기존 시설 시너지/중복**:
  - 금고(방치 보상 상한 +960G) → **무관**. 방치 vs 능동.
  - 대장간(장비 강화, stub) → **무관**.
- **패시브 차별화**: 상인 연합 패시브 = `quest_reward_multiplier` (호위 보상 +12%, escort 한정) + `idle_reward_bonus` (방치 +10%). 시설 = **모든 quest_type 골드 +20%** → escort에는 패시브와 가산되어 +32% 효과, 다른 type에도 +20%. 이는 "상인 연합 가입 = 골드 수급 우월"이라는 정체성을 패시브-시설 양쪽으로 강화.

---

##### 3. 전사 길드 — **무기고** (Expensive, α=0.3)

> 전설의 무구를 보관하고 출정 직전에 빌려주는 보관소. 무기고에서 무기를 들고 나간 용병은 평소보다 더 강력한 일격을 노린다.

- **효과**: raid/hunt 퀘스트에서 **대성공 판정 확률 가산** (성공 → 대성공 변환 가산률). 신규 효과 타입 `great_success_rate_bonus`.
  - max_effect 후보: 0.25 (Lv15에서 raid/hunt 한정 +25%p 대성공 확률)
- **카테고리 근거**: 대성공 = 보상 2배, 직접 골드 영향. Expensive 합당.
- **기존 시설 시너지/중복**:
  - 대장간(장비 효과, stub) → **다른 차원**. 대장간이 "장비 자체"라면 무기고는 "결과 판정 변환".
  - 게시판(고보상 확률) → **다른 차원**. 게시판이 "고보상 퀘스트 출현"이라면 무기고는 "퀘스트 결과 등급 상승".
- **패시브 차별화**: 전사 길드 패시브 = `quest_success_rate_bonus` (raid/hunt 성공률 +5%). 시설 = **대성공 확률 가산** (성공률과 다른 차원). 패시브가 성공/실패 분기점을 옮기는 것이라면 시설은 성공 안에서 대성공 비율을 높임.

---

##### 4. 도둑 길드 — **암시장** (Standard, α=0.3)

> 길드 지하의 거래소. 공식 시장에서는 거절당할 거래가 여기서 이뤄진다. 트레잇 시술과 모집 알선의 수수료를 후려친다.

- **효과**: **트레잇 삭제 비용 + 골드 모집 비용** 비례 할인. 신규 효과 타입 `service_fee_discount` (대상: trait_delete, recruit_paid).
  - max_effect 후보: 0.50 (Lv15에서 -50% 비용)
- **카테고리 근거**: 간접 골드 절감 효과. 직접 수익원 아님 → Standard.
- **기존 시설 시너지/중복**:
  - 의무실(트레잇 삭제 해금 — Lv2/Lv4) → **보완**. 해금은 의무실, 비용 할인은 암시장.
  - 주점(모집 확률 향상) → **보완**. 주점이 결과 품질, 암시장이 비용.
- **패시브 차별화**: 도둑 길드 패시브 = `travel_event_mitigation` (이동 골드 손실 -30%) + `investigation_success_rate_bonus` (조사 +5%). 시설 = **고정 지출 항목 비용 할인**. 패시브가 "잃지 않기"라면 시설은 "덜 쓰기" — 같은 결과(잔고 보존)를 다른 경로로.

##### 5. 마탑 연합 — **마법 연구실** (Premium, α=0.3)

> 마탑이 후원하는 마술적 분석 시설. 용병들의 행동 패턴을 마법 도구로 기록·분석하여 깊은 통찰을 끌어낸다.

- **효과**: 모든 퀘스트 완료 시 **행동 지표(stats) 누적 속도 가산**. 신규 효과 타입 `behavior_stat_accumulation_bonus`.
  - max_effect 후보: 0.50 (Lv15에서 +50% 누적)
  - 즉, raid_count/success_count 등 23개 행동 지표가 1회 가산될 때 ×1.5로 가산. 트레잇 획득 임계값 도달 속도 ↑.
- **카테고리 근거**: 정성 효과(트레잇 시스템 영향). Premium 합당.
- **기존 시설 시너지/중복**:
  - 훈련소(XP +80%) → **다른 차원**. XP는 레벨업 속도, 행동 지표는 트레잇 획득 속도.
  - 연구소(특수 트레잇 해금, stub) → **다른 차원**. 해금 vs 누적 속도.
  - **트레잇 학습 부스트(M3 변형 섹터 보상, 분당 ×2)** → **이중 적용 방지**. 부스트 활성 중에는 마법 연구실 효과 무시 (Phase 4 명세 시 반영). 또는 곱셈 결합(연구실 ×1.5 + 부스트 ×2 = 최종 ×3) — 이는 페이즈 2 밸런스 결정.
- **패시브 차별화**: 마탑 연합 패시브 = `quest_success_rate_bonus` (탐험 +8%) + `trait_acquisition_condition_relief` (조건 -10% 완화). 시설 = **누적 속도 가산**. 패시브가 "임계값 자체를 낮춤"이라면 시설은 "임계값 도달 속도를 높임". 두 효과가 곱셈 결합되어 트레잇 획득 가속도 합산.

##### 6. 태양 교단 — **태양 신전** (Premium, α=0.3)

> 출정 전 신성한 가호를 받는 신전. 태양신의 축복은 위기의 순간 용병을 죽음의 문턱에서 끌어올린다.

- **효과**: 부상 발생 시 **확률적으로 부상 무효화** (신성 가호). 신규 효과 타입 `injury_prevention_chance`.
  - max_effect 후보: 0.40 (Lv15에서 40% 확률로 부상 무효)
  - 운영: 퀘스트 결과가 "실패(부상)"로 판정될 때 시설 효과 확률 굴림 → 성공 시 부상 상태 미적용 (단, 결과 자체는 실패 유지).
- **카테고리 근거**: 정성 효과 (시설 비용/시간 절감보다 강력하지만 골드 직접 영향 아님). Premium.
- **기존 시설 시너지/중복**:
  - 의무실(회복 시간 -70%), 야전병원(부상 시간 -35%) → **다른 차원**. 의무실/야전병원이 "부상 후 시간 단축"이라면 태양 신전은 "부상 자체 회피". 부상이 걸리지 않으면 의무실/야전병원이 적용될 일이 없음 — 의무실/야전병원의 가치를 떨어뜨릴 수 있다는 우려가 있으나, 60% 확률로는 여전히 부상 발생하므로 둘 다 의미.
  - 방어시설(여행 이벤트 피해 -50%) → **다른 차원**. 방어시설이 이동 중 피해라면 태양 신전은 퀘스트 결과 부상.
- **패시브 차별화**: 태양 교단 패시브 = `quest_success_rate_bonus` (escort +8%) + `recovery_time_reduction` (부상 회복 -15%). 시설 = **부상 자체 회피**. 패시브가 "부상 시간 줄이기"라면 시설은 "부상이 안 걸리기" — 정체성 일관성 (수호/가호).

##### 7. 균형 감시자 — **감시 초소** (Premium, α=0.4)

> 결사가 비밀리에 운영하는 정찰 거점. 한 번의 조사로 여러 단서를 동시에 포착할 수 있는 정밀 관찰 장비를 비치한다.

- **효과**: 지역 조사 1회 완료 시 **추가 발견 슬롯 노출 확률** 가산. 신규 효과 타입 `investigation_extra_discovery_chance`.
  - max_effect 후보: 0.50 (Lv15에서 50% 확률로 추가 발견 1개)
  - 운영: 조사 성공으로 발견 트리거가 활성화될 때 시설 효과 확률 굴림 → 성공 시 추가 발견 슬롯 1개 더 트리거 (가용 후보가 있을 경우).
- **카테고리 근거**: 정성 효과 (조사 시스템 영향). 발견은 게임 진행 핵심이라 Premium.
- **기존 시설 시너지/중복**:
  - 연구소(조사 효율 +60%, stub) → **다른 차원**. 연구소가 "성공률/시간"이라면 감시 초소는 "발견 다중화".
  - 정보망(퀘스트 슬롯) → **무관**.
- **패시브 차별화**: 균형 감시자 패시브 = `quest_success_rate_bonus` (전체 +3%). 시설 = **조사 다중 발견** → 패시브와 무관 영역. 정체성 일관성 (감시/관찰).

##### 8. 금지된 서고 — **금서 열람실** (Expensive, α=0.3)

> 봉인된 마법서들이 보관된 비밀 열람실. 금서를 열람한 용병은 트레잇 진화 시 일반인이 접근할 수 없는 갈래를 본다.

- **효과**: 후천 트레잇 진화 시 **추가 분기 옵션 노출 확률** 가산. 신규 효과 타입 `trait_evolution_extra_branch_chance`.
  - max_effect 후보: 0.50 (Lv15에서 50% 확률로 추가 진화 분기 노출)
  - 운영: `trait_transitions` 또는 `trait_combo_evolutions`에서 정상 분기를 결정한 후, 시설 효과 확률 굴림 → 성공 시 "금지" 태그 분기 1개 추가 노출 (별도 데이터 풀 — 페이즈 1 #4 트레잇 컨셉에서 정의).
- **카테고리 근거**: 강력한 트레잇 영향 (희귀 분기 노출). Expensive.
- **기존 시설 시너지/중복**:
  - 연구소(특수 트레잇 해금, stub) → **다른 차원**. 연구소가 "해금"이라면 서고는 "분기 추가 노출".
- **패시브 차별화**: 금지된 서고 패시브 = `trait_evolution_condition_relief` (조건 -15% 완화). 시설 = **추가 분기 노출**. 패시브가 "더 빨리 진화"라면 시설은 "더 다양하게 진화". 정체성 일관성 (금단의 지식 = 다양성).

##### 9. 뿌리의 맹세단 — **자연 제단** (Premium, α=0.3)

> 자연의 정령들에게 올리는 제단. 죽음을 앞둔 용병에게 마지막 자비를 베풀어 부상으로 강등시킨다.

- **효과**: 대실패로 인한 사망 판정 시 **부상으로 강등 확률** 가산. 신규 효과 타입 `death_to_injury_downgrade_chance`.
  - max_effect 후보: 0.35 (Lv15에서 35% 확률로 사망 → 부상 강등)
  - 운영: 퀘스트 결과 "대실패 + 사망률 굴림 → 사망 판정"이 발생할 때 시설 효과 확률 굴림 → 성공 시 사망 대신 부상(난이도 × 10분) 적용.
- **카테고리 근거**: 영구 손실 방지 (사망 = 영구 제거). 매우 강력하지만 사용 빈도 낮음 → Premium.
- **기존 시설 시너지/중복**:
  - 방어시설(여행 이벤트 피해 -50%) → **다른 차원**. 방어시설이 이동 중이라면 자연 제단은 퀘스트 결과 사망.
  - 의무실/야전병원 → **다른 차원**. 회복 시간 vs 사망 판정.
  - **태양 신전(부상 무효 40%)** → **상호 배제 검토 필요**. 태양 신전이 "실패 → 부상" 단계에서 부상을 무효화한다면, 자연 제단은 "대실패 → 사망" 단계에서 사망을 부상으로 강등. 둘은 다른 결과 등급에 적용되므로 중복 없음. 단, 사망이 부상으로 강등된 후 태양 신전이 그 부상을 무효화하는 이중 보호 시나리오는 가능 — 페이즈 2/4 검토.
- **패시브 차별화**: 뿌리의 맹세단 패시브 = `travel_event_mitigation` (이동 피해 -40%) + `recovery_time_reduction` (회복 -15%). 시설 = **사망 → 부상 강등**. 패시브가 "안 다치기"라면 시설은 "안 죽기" — 정체성 일관성 (자연의 자비/생명 보존).

##### 10. 황혼 공학회 — **개조 실험실** (Expensive, α=0.3, **능동형 시설**)

> 결사의 핵심 연구 거점. 후천 트레잇을 강제로 떼어내고 다른 트레잇을 이식한다. 성공하면 인간의 한계를 뛰어넘지만 실패하면 모든 것을 잃는다.

- **효과**: **능동 액션 시설** — 사용 시 후천 트레잇 1개를 골드로 강제 교체. 시설 레벨이 올라갈수록 (1) 성공률 ↑, (2) 쿨다운 ↓.
  - max_effect 후보:
    - 성공률: 0.85 (Lv15에서 85%, 실패 시 트레잇 소실)
    - 쿨다운: 7일 → 1일 (Lv15)
  - 신규 효과 타입: `trait_replacement_action`. UI에 "사용" 버튼 노출, 사용 시 골드 차감 + 트레잇 선택 + 결과 확률.
- **카테고리 근거**: 능동 사용 + 강력한 변경 능력. Expensive.
- **기존 시설 시너지/중복**:
  - 의무실(트레잇 삭제 해금) → **보완**. 삭제(슬롯 비우기) vs 교체(다른 트레잇으로).
- **패시브 차별화**: 황혼 공학회 패시브 = `facility_cost_reduction` (-10%) + `facility_effect_bonus` (+5%). 시설 = **능동 트레잇 교체**. 패시브가 "시설을 더 잘 쓰기"라면 시설은 "기술로 한계 돌파". 정체성 일관성 (인체 개조).
- **주의**: 능동형 시설은 다른 시설들과 효과 곡선 구조가 다르다. 페이즈 2/4에서 별도 처리 필요 (effect_json 스키마 확장).

##### 11. 심층 망치단 — **룬 대장간** (Expensive, α=0.3)

> 드워프 장인이 직접 운영하는 비밀 대장간. 길드 장비에 룬을 새겨 효과를 증폭시킨다. 일반 대장간과는 차원이 다른 정밀 단련.

- **효과**: M3에서 도입된 **길드 장비** (chain quest 보상)의 효과 배율 가산. 신규 효과 타입 `guild_equipment_effect_bonus`.
  - max_effect 후보: 0.30 (Lv15에서 길드 장비 효과 +30%)
  - 운영: 장착된 길드 장비의 effect_json 적용 시 ×(1+value) 배율.
- **카테고리 근거**: 직접 전투력 영향 (장비 효과 강화). Expensive.
- **기존 시설 시너지/중복**:

- **패시브 차별화**: 심층 망치단 패시브 = `facility_cost_reduction` (시간 -20%). 시설 = **장비 효과 증폭**. 패시브가 "더 빨리 짓기"라면 시설은 "더 강한 단련" — 정체성 일관성 (장인 정신).
- **의존성**: 길드 장비 시스템 (M3에서 chain quest 보상으로 도입됨, `Mercenary` 길드 장비 슬롯 — 이미 구현됨). M4 시점에 호환 가능.

##### 12. 화산 심장단 — **용암 단련장** (Standard, α=0.3)

> 활화산의 열기로 갑주를 단련하는 노천 시설. 출정 직전 갑주를 두드려 무게를 줄이고, 파견 비용을 깎아낸다.

- **효과**: 모든 퀘스트의 **파견 비용** (난이도 × 시간 비례 골드) 비율 감소. 신규 효과 타입 `dispatch_cost_reduction`.
  - max_effect 후보: 0.30 (Lv15에서 파견비 -30%)
- **카테고리 근거**: 간접 골드 절감 (직접 수익원 아님). Standard.
- **기존 시설 시너지/중복**:
  - 금고(방치 보상) → **무관**.
  - 이동수단(이동 시간 -40%) → **무관**.
- **패시브 차별화**: 화산 심장단 패시브 = `quest_reward_multiplier` (raid 보상 +15%). 시설 = **파견 비용 감소** (모든 quest_type). 패시브가 "보상 ↑"이라면 시설은 "지출 ↓" — 결합 시 raid 순수익 폭발적 증가. 정체성 일관성 (파괴적 효율 = 모든 비효율 태움).

##### 13. 혈계 귀족회 — **혈계 도서관** (Premium, α=0.3)

> 혈통 가문이 대대로 축적한 가문서가 보관된 도서관. 용병들이 기록을 읽으며 혈통의 잠재력을 각성시킨다.

- **효과**: 용병 레벨업 시 **추가 스탯 보너스 배율** 가산. 신규 효과 타입 `levelup_stat_bonus_multiplier`.
  - max_effect 후보: 0.40 (Lv15에서 레벨업 시 추가 스탯 +40%)
  - 운영: 레벨업 시 직업/티어별 기본 스탯 증가량을 ×(1+value)로 부스트.
- **카테고리 근거**: 정성 효과 (장기 캐릭터 성장 영향). Premium.
- **기존 시설 시너지/중복**:
  - 훈련소(XP +80%) → **보완**. 훈련소가 "레벨업 속도"라면 혈계 도서관은 "레벨업 가치". 두 시설 결합 시 후반 용병 강도 폭발.
- **패시브 차별화**: 혈계 귀족회 패시브 = `recruitment_tier_boost` (T4~T5 +4%). 시설 = **레벨업 스탯 부스트**. 패시브가 "고급 용병 영입"이라면 시설은 "용병 자체 강화" — 정체성 일관성 (혈통 = 잠재력).

##### 14. 송곳니 결사 — **부족 훈련장** (Premium, α=0.3)

> 부족이 모여 함께 훈련하는 야외 훈련장. 큰 무리로 출정할 때 결사의 옛 전통이 되살아난다.

- **효과**: **5명 이상 파티 파견 시 추가 성공률 가산**. 신규 효과 타입 `quest_success_rate_bonus_party_size_facility` (패시브와 별도 풀).
  - max_effect 후보: 0.10 (Lv15에서 5명+ 파티 시 +10%p, 시설 별도 풀이라 패시브 +20%p 공유 상한과 무관)
- **카테고리 근거**: 성공률 영향 (강력하지만 5명+ 조건부). Premium.
- **기존 시설 시너지/중복**:
  - 주둔지(용병 상한 +20명) → **간접 시너지**. 주둔지가 "5명+ 파티 가능 인원 확보"라면 부족 훈련장은 "5명+ 파티 효과 강화".
- **패시브 차별화**: 송곳니 결사 패시브 = `quest_success_rate_bonus_party_size` (3명 이상 +8%). 시설 = **5명+ 추가 가산**. 패시브가 "다수 파견 권장"이라면 시설은 "더 큰 무리 권장". 둘은 인원 임계값이 다르고 시설 효과는 별도 풀이라 합산 시 5명+ 파티에 +18%p 적용 (단, 패시브는 +20%p 공유 상한 내, 시설은 별도).
- **주의**: 시설 효과를 패시브 공유 상한에서 분리하는 이유 = 만약 합산 풀이면 후반에 +20%p 상한에 도달하여 시설 효과가 무의미해짐. 시설 별도 풀로 처리하면 시설 투자에 의미 부여.

## 미해결 사항 (페이즈 2/4에서 결정)

1. **건설 큐 공유 vs 분리**: 본 문서는 공유 권장. 페이즈 4 명세에서 최종 결정. 분리 시 `UserData`에 추가 Hive 필드 필요.
2. **개조 실험실(10) 능동형 시설의 effect_json 스키마**: 다른 13종은 정적 보정값이지만 이것만 능동 액션. 페이즈 2 밸런스 + 페이즈 4 데이터 모델 확장 필요.
3. **태양 신전 + 자연 제단 이중 보호 시나리오**: 자연 제단이 사망을 부상으로 강등 → 태양 신전이 그 부상을 무효화하는 두 단계 보호가 가능한지. 페이즈 4 명세에서 처리 순서 결정.
4. **마법 연구실(5) + 트레잇 학습 부스트(M3) 결합**: 곱셈 vs 부스트 우선. 페이즈 2 밸런스에서 확정.
5. **시설 효과 + 패시브 공유 상한**: 송곳니 결사 부족 훈련장은 별도 풀로 결정했지만, 다른 효과(예: 무기고의 대성공 확률)도 별도 풀인지 패시브와 공유인지 페이즈 4에서 명문화 필요.

## data-generator 지시사항

> 본 기획서로부터 페이즈 3 #1 (`faction-facility` 데이터 생성)이 호출된다.

- **대상 타입**: `faction-facility` (data-generator 신규 타입 — 타입 스펙 작성 필요: `Docs/superpowers/data-generator/types/faction-facility.md`)
- **대상 테이블**: `faction_facilities` (Supabase 신규)
- **생성 수량**: 14행 (세력 1:1 매핑)
- **컬럼 구조**:
  ```
  id            TEXT PK              -- 'fac_facility_<faction_id_short>'
  faction_id    TEXT FK → factions.id
  name          TEXT                 -- 한글
  description   TEXT                 -- 한글, 1~2문장
  category      TEXT                 -- 'standard' | 'premium' | 'expensive'
  base_cost     INTEGER              -- 500 / 700 / 1000
  cost_multiplier REAL               -- 1.5 (전 시설 공통)
  lv1_cost      INTEGER              -- base_cost * 0.1
  lv2_cost      INTEGER              -- base_cost * 0.2
  base_time     INTEGER              -- 25 (분, 전 시설 공통)
  time_multiplier REAL               -- 1.45 (전 시설 공통)
  lv1_time      INTEGER              -- 5 (분)
  lv2_time      INTEGER              -- 10 (분)
  max_level     INTEGER              -- 15 (전 시설 공통)
  max_effect    REAL                 -- 본 문서 섹션 3 표 참조
  alpha         REAL                 -- 0.3 또는 0.4 (개조 실험실은 능동형 — 별도 처리)
  effect_type   TEXT                 -- 본 문서 섹션 3 표 참조
  effect_json   JSONB                -- 효과 타입별 파라미터 (페이즈 4 명세 후 확정)
  is_active_action BOOLEAN           -- false (개조 실험실만 true)
  ```
- **톤/세계관 가이드**:
  - 세력 정체성에 직결되는 한국어 시설명 (본 문서 섹션 2 참조)
  - 설명은 1~2문장, 시설 위치/용도/세력과의 연결 명시
  - 외래어 남발 금지 (한국어 위주)
- **구조적 제약**:
  - faction_id는 14개 세력 ID와 1:1 매칭
  - category 분포: Expensive 5 / Premium 7 / Standard 2
  - max_effect 값은 본 문서 섹션 3 표 그대로 (페이즈 2 밸런스 후 갱신 가능)
- **수치 출처**: `Docs/balance-design/[balance]20260428_faction-facility-values.md` (페이즈 2 #1 산출물)
- **특수 요구**:
  - 개조 실험실(10)은 능동형 시설 → `is_active_action=true`, effect_json 스키마 별도
  - effect_json은 페이즈 4 명세에서 효과 타입별 스키마 확정 후 채움 (페이즈 3 시점에는 스키마 placeholder)
