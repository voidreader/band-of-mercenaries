## 개요

본 문서는 M7 마일스톤의 **5~8시간 누적 플레이 흐름**을 분 단위로 명세하여 M7 종료 조건(주요 사건 2+, 제작 목표 2+, 지역 상태 변화 2+) 충족을 검증한다. M4 페이즈 1 #5 "초반 2시간 플레이 흐름"의 연장으로 작성된다. 즉, 본 문서는 **120분 ~ 480분 (= 8시간)** 구간을 다룬다. M7 페이즈 1 #1·#2·#3의 모든 결정을 시간축에 매핑하고, 이동 화면 UI 컨셉 + `region_adjacency` 도입 결정 + 3 페르소나(적극·평균·보수) 시간선 비교를 포함한다.

본 문서가 결정하는 4가지:

1. **분 단위 시나리오** — 120분 ~ 480분, 30분 단위 sub-window로 7리전 사건 진행 + dangerScore 변화 + unlockedFlags 누적 + 인프라 단계 전이 매핑
2. **중간 목표 2개 / 최종 목표 1개** — 신규 유저가 어떤 마일스톤을 어느 시점에 체감하는가
3. **이동 화면 UI 컨셉** — 7리전 생활권 그룹화 + dangerLevel 시각 + 인프라 단계 시각 (페이즈 4 #3 명세 입력)
4. **`region_adjacency` 도입 결정 확정** — 페이즈 1 #1 5절의 옵션 A를 본 문서에서 채택, 7리전 × 평균 3인접 ≈ 약 21행 매트릭스 컨셉

본 문서는 **이상적 시나리오**(M4 페이즈 1 #5 패턴)다. 플레이어 이탈·재방문·실패 분기는 페이즈 2 #1·#3 시뮬레이션에 위임한다.

**핵심 원칙 3가지**:
1. **이동 = 명시적 선택** — "할 게 없어서 이동"이 아니라 "재료/사건/위험도를 위해 이동"이 되도록 매 30분마다 명확한 외출 목적을 매핑
2. **누적 플레이 보장** — 5시간 시점에 M7 종료 조건 모두 충족 가능, 8시간 시점에 모든 사건 + Tier 4 도달 (적극 플레이 시)
3. **M4 흐름과 매끄러운 연결** — 0~120분(M4) → 120~480분(M7) 사이 끊김 없는 동기 부여. 120분 시점에 폐광 step 6 마무리 + region 31·127 첫 외출 시작

```sql
CREATE TABLE region_adjacency (
  id SERIAL PRIMARY KEY,
  from_region INTEGER NOT NULL REFERENCES regions(region),

CREATE INDEX idx_region_adjacency_from ON region_adjacency(from_region);
```

**MovementService 변경 의사 코드** (페이즈 4 #3 명세 입력):

```dart
int calculateDistance(int fromRegion, int toRegion, int fromSector, int toSector) {
  // 동일 region 이동
  if (fromRegion == toRegion) {
    return (toSector - fromSector).abs();
  }

  // 인접 그래프 조회 (region_adjacency 테이블 활용)
  final adjacency = _staticData.regionAdjacency[fromRegion]?[toRegion];
  if (adjacency != null) {
    return adjacency + (toSector - fromSector).abs();
  }

  // fallback: 기존 |ID 차이| 계산식 (인접성 없는 리전 간 이동)
  return (toRegion - fromRegion).abs() + (toSector - fromSector).abs();
}
```

**광장 이정표 효과 (M7 페이즈 1 #3 Tier 2 효과) 통합**:

```dart
// Movement 거리 계산 후 광장 이정표 효과 적용
int finalDistance = calculateDistance(...);
final infraTier = regionStateRepo.getOrCreate(3).currentInfrastructureTier;
if (infraTier >= 2 && (fromRegion == 3 || toRegion == 3)) {
  // 더스트빌 출발 또는 도착 시 -10%
  finalDistance = (finalDistance * 0.9).round();
}
return finalDistance;
```
