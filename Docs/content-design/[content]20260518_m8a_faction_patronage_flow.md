## 개요

본 문서는 M8a 대표 3세력의 평판 흐름과 갈등 구조를 정의한다. 기존 `FactionJoinService`의 규칙을 유지하면서, 가입 전 접촉, 후원 상태, 축소 상점, 세력 지명 의뢰가 자연스럽게 이어지도록 한다.

M8a의 핵심 결정은 **가입보다 후원을 먼저 체감하게 한다**는 것이다. 플레이어는 곧바로 세력 본부에 소속되는 것이 아니라, 생활권 접촉점을 통해 "이 세력이 우리 용병단을 눈여겨본다"는 상태를 먼저 경험한다.

## 레퍼런스 분석

| 레퍼런스 | 차용 메커니즘 | 적용 방식 |
|---------|--------------|----------|
| Fallout: New Vegas — 세력 평판 | 소속 전에 평판과 접촉 태도가 먼저 바뀐다 | M8a는 `noticed`와 `patronage` 상태를 도입한다 |
| Crusader Kings 3 — 후원자 관계 | 정식 봉신 관계 전에도 후원·호의·계약 관계가 존재한다 | 세력 가입 전에도 지명 의뢰와 일부 상점이 열린다 |
| Darkest Dungeon — 마을 이벤트 | 특정 성과가 다음 행동 선택지를 잠깐 바꾼다 | 지역 안정화 직후 세력 반응 텍스트와 의뢰가 열린다 |
| Sunless Skies — 세력별 적대와 중립 | 한쪽의 신뢰가 다른 쪽의 경계를 만든다 | 상인 연합과 전사 길드 갈등을 낮은 강도로 먼저 노출한다 |

## 상태 흐름

### M8a 세력 관계 단계

기존 평판 수치(-100~100)는 유지한다. M8a는 UI와 노출 정책을 위해 관계 단계를 해석한다.

| 단계 | 조건 | 의미 | 가능한 행동 |
|------|------|------|-------------|
| 미접촉 | 접촉점 조건 미충족 | 세력이 용병단을 모른다 | 도감 기본 정보만 확인 |
| 주목 | 접촉점 조건 충족, 평판 0~10 | 세력이 용병단을 관찰한다 | 반응 텍스트, 1회성 접촉 이벤트 |
| 후원 | 평판 1~10 또는 접촉점 방문 완료 | 가입 전 소형 의뢰를 맡긴다 | 세력 지명 의뢰 일부, 상점 1~2개 |
| 가입 | 기존 `canJoin` 조건 충족 후 가입 | 기존 세력 가입 상태 | 패시브, 상점 3~6개, 평판 100 상한 |
| 신뢰 | 가입 + 평판 31 이상 | 세력의 실질적 동업자 | 고급 지명 의뢰, 전용 보상 |
| 핵심 | 가입 + 평판 61 이상 | M8a에서는 1개 상품 또는 보상만 노출 | M9 확장 여지 |
| 적대 | 평판 < 0 | 세력이 의뢰를 맡기지 않는다 | 적대 반응만 표시 |

### 후원 상태의 역할

후원 상태는 정식 가입 이전의 완충 단계다. M8a에서 후원 상태는 별도 Hive 필드 없이 계산형 상태로 시작할 수 있다.

```text
후원 상태 =
  접촉점 해금 조건 충족
  AND reputation >= 1
  AND joined == false
  AND reputation <= 10
```

이 방식은 기존 `FactionJoinService.maxReputationBeforeJoin = 10`과 호환된다. 페이즈 4에서 재방문 반응이나 방문 이력을 추가하려면 `FactionState` HiveField 6 이후에 `contactVisitedAtById` 같은 필드를 검토한다.

## 대표 세력별 평판 흐름

### 모험가 길드

모험가 길드는 가장 먼저 접촉할 수 있는 세력이다. "가입"보다 "기록 등록"을 먼저 제공한다.

| 단계 | 조건 | 반응 |
|------|------|------|
| 주목 | 폐광 재개방 또는 인프라 Tier 2 | 기록원이 더스트빌에 도착한다 |
| 후원 | 평판 1~10 | 기록 검증 의뢰 1~2개 노출 |
| 가입 | 랭크 E + 평판 > 0 | 기존 패시브 `explore` 보상 활성 |
| 신뢰 | 평판 31 이상 | 고급 지도 보정 의뢰, `길드 장부에 오른 자` 보상 후보 |

모험가 길드는 갈등 압력이 낮다. 기존 충돌 세력은 도둑 길드이지만 M8a에는 도둑 길드를 대표 세력으로 포함하지 않으므로 직접 갈등은 반응 텍스트에서만 암시한다.

### 상인 연합

상인 연합은 M7 인프라 성장과 가장 강하게 연결된다. 외래 좌판을 통해 후원 상태에 진입한다.

| 단계 | 조건 | 반응 |
|------|------|------|
| 주목 | 인프라 Tier 3 또는 `region_127_nomad_friendly` | 케일이 상인 연합 대리인임을 밝힌다 |
| 후원 | 평판 1~10 | 변방 해안 호송, 도적길 교역 의뢰 노출 |
| 가입 | 랭크 E + 평판 > 0 | 기존 패시브 `escort` 보상 + 방치 보상 활성 |
| 신뢰 | 평판 31 이상 | 상단 보증서, 교역 인장 장식 보상 후보 |

상인 연합은 전사 길드와 충돌한다. M8a에서는 강제 적대까지 몰아가지 않고, 다음 정책을 적용한다.

- 상인 연합 가입 상태에서 전사 길드 후원 의뢰를 완료하면 상인 연합 평판 -1을 검토한다.
- 전사 길드 가입 상태에서 상인 연합 후원 의뢰를 완료하면 전사 길드 평판 -1을 검토한다.
- 실제 자동 탈퇴와 -100 적대는 기존 `FactionJoinService`의 가입 충돌 처리에서만 발생한다.

### 전사 길드

전사 길드는 가장 늦게 접촉한다. 전투 성과를 증명한 뒤 의뢰를 제안한다.

| 단계 | 조건 | 반응 |
|------|------|------|
| 주목 | `region_9_giant_beast_killed` 또는 region 38 접근 | 붉은 창의 벨란이 결투 표식을 남긴다 |
| 후원 | 평판 1~10 | 외곽 숲 재검증, 요새 결투장 의뢰 노출 |
| 가입 | 랭크 D + 평판 > 0 | 기존 패시브 `raid/hunt` 성공률 활성 |
| 신뢰 | 평판 31 이상 | `결투 표식을 받은 자`, 붉은 창 보상 후보 |

전사 길드는 플레이어에게 "위험한 의뢰를 받아 더 큰 보상을 얻을 것인가"라는 선택지를 준다. 상인 연합과의 갈등은 보상 방향이 다르므로 충분히 체감된다.

## 갈등 구조

### M8a에서 다루는 갈등

| 갈등 | 적용 강도 | 설명 |
|------|----------|------|
| 상인 연합 vs 전사 길드 | 중간 | 기존 충돌 구조를 M8a의 대표 갈등으로 사용한다 |
| 모험가 길드 vs 도둑 길드 | 낮음 | 도둑 길드가 M8a 대표 세력이 아니므로 풍문과 반응 텍스트로만 사용한다 |
| 세력 vs 지역 유목민 | 낮음 | 유목민은 독립 세력이 아니라 상인 연합의 협력자로 처리한다 |
| 세력 vs 더스트빌 | 낮음 | 세력은 마을을 장악하지 않고 후원한다 |

### 충돌 처리 원칙

1. 정식 가입 충돌은 기존 정책을 유지한다.
2. 후원 상태끼리는 서로 배타적이지 않다.
3. 후원 의뢰 완료로 평판 -100을 만들지 않는다.
4. 갈등은 처음에는 반응 텍스트와 소량 평판 변화로만 표현한다.
5. 플레이어가 어느 세력에 가입할지 결정하기 전까지 선택지를 과하게 닫지 않는다.

## 평판 획득과 소비

### 획득 경로

| 경로 | 가입 전 | 가입 후 | 비고 |
|------|---------|---------|------|
| 세력 태그 일반 의뢰 | +1~2, 상한 10 | +1~2 | 기존 M1 유지 |
| 세력 지명 의뢰 | +2~4, 상한 10 | +4~8 | 페이즈 2에서 확정 |
| 세력 전용 의뢰 | 가입 전 불가 | +5~10 | 기존 M1 유지 |
| 접촉점 1회 이벤트 | +1 | 없음 | 세력 도착 체감용 |
| 전투 보고서 중요 장면 | 없음 | +1 보너스 후보 | M8a에서는 보류 가능 |

### 소비 또는 잠금 경로

M8a에서는 평판을 직접 소비하지 않는다. 상점은 평판 조건만 요구한다. 평판 소비형 상품은 장기적으로 흥미롭지만, 초반 세력 학습 단계에서는 "샀더니 평판이 줄었다"는 경험이 혼란스럽다.

## 세력 후원과 지역 상태 연결

| 지역 상태 플래그 | 반응 세력 | 결과 |
|-----------------|----------|------|
| `region_31_bandits_cleared` | 상인 연합 | 도적길 교역 의뢰와 상점 재료 할인 후보 |
| `region_127_nomad_friendly` | 상인 연합 | 외래 좌판 확장, 변방 해안 호송 의뢰 |
| `region_9_giant_beast_killed` | 전사 길드 | 강적 재검증 의뢰, 전투 보고서 강제 생성 |
| `region_38_ironbound_pact_completed` | 전사 길드, 모험가 길드 | 요새 기록 의뢰와 결투 의뢰 동시 후보 |
| `region_10_windrunner_chain_completed` | 모험가 길드 | 풍신 숲 지도 보정 의뢰 |

이 연결은 M8a의 핵심이다. 세력이 단순히 화면에서 나타나는 것이 아니라, 플레이어가 M7에서 해결한 사건을 근거로 반응한다.

## 현재 시스템과의 연관

| 시스템 | 유지 | 확장 후보 |
|--------|------|----------|
| `FactionJoinService` | 가입 조건, 충돌 가입, 평판 클램프 유지 | 없음 |
| `FactionState` | 평판·가입·단서 유지 | 접촉점 방문 이력, 후원 상태 캐시 |
| `RegionState` | unlockedFlags와 infrastructureTier 사용 | 없음 |
| `NamedHookEvaluator` | title/achievement/flagship hook 유지 | `region_flag`, `faction_reputation`, `faction_contact` hook 추가 |
| `ActivityLog` | 세력 평판 획득 로그 유지 | 접촉점 해금 로그 추가 후보 |

## 구현 우선순위 제안

| 우선순위 | 항목 | 이유 |
|----------|------|------|
| 높음 | 후원 상태 계산 규칙 | 가입 전 세력 경험을 닫는 핵심이다 |

## data-generator 지시사항

- **대상 타입**: 접촉점·반응 텍스트 타입 스펙 필요
- **대상 테이블**: 신규 후보 `faction_reaction_texts`
- **생성 수량**: 30~50개
- **톤/세계관 가이드**:
  - 주목 단계: "소문을 들었다"보다 구체적 사건을 언급한다.
  - 후원 단계: 계약·기록·표식처럼 세력별 매개물을 사용한다.
  - 갈등 단계: 직접 비난보다 이해관계의 차이를 드러낸다.
- **구조적 제약**:
  - 후원 상태는 가입이 아니다.
  - 반응 텍스트는 세력당 최소 8개 이상 작성한다.
  - 지역 상태 플래그 5종 이상을 조건으로 사용한다.
- **수치 출처**: 페이즈 2 #2에서 평판 보상량과 갈등 페널티를 확정한다.
