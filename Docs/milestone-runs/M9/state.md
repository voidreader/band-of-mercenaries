## 로드맵 요구사항 요약

M9 "완성과 균형"은 M4~M8.5에서 검증된 재미 축을 기준으로 데이터를 확장하고, stub 기능을 실제 동작으로 연결하며, 경제·전투·성장 밸런스를 통합 조정한다. 기존 v2의 "전체 데이터 채우기"는 이 시점에서만 재개한다.

핵심 목표는 검증된 구조를 확장 가능한 패턴으로 굳히는 것이다. 일반 의뢰까지 전투 시뮬레이션 적용 범위를 넓히고, 직업군·티어·일부 고유 직업 스킬을 확장하며, 대장간·연구소·게시판 stub을 실제 시스템과 연결한다.

선행 의존성: M8.5 완료.

## 포함 범위

M9는 다음 범위를 포함한다.

- 생활권 단위 데이터 확장.
- 일반 의뢰까지 턴 기반 전투 시뮬레이션 적용 범위 확장.
- 직업군 대표 스킬의 티어별·직업별 확장.
- 트레잇, 장비, 칭호와 스킬 발동률 또는 효과 연결.
- 대장간, 연구소, 게시판 stub 시설 연동.
- 티어 업그레이드 재도입 여부 재검토.
- 경제, 전투, 성장, 제작, 위업, 세력, 경쟁 밸런스 통합 검증.
- operation-bom 데이터 검증 대시보드 또는 검증 쿼리.
