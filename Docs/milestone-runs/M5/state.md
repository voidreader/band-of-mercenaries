## 로드맵 요구사항 요약

M5는 단순 완제품 드랍 중심의 보상 구조를 "여러 활동에서 재료를 모아 장비와 아티팩트를 완성하는 구조"로 바꾼다. 핵심 장비/아티팩트는 파견·조사·엘리트·이동 선택지·고정 사건에서 얻는 재료를 조합해 완성한다(완제품 드랍은 병행 유지). 대표 제작 목표는 ① 낡은 용병단 깃발 복원 ② 광부의 단검 또는 수습대장의 검 ③ 폐광의 유물 조각 3종. 스키마: `items` 테이블 category/slot/effect 확장 + `crafting_recipes` 신규 테이블(또는 JSONB MVP) + `CraftingService`. UI: 인벤토리 재료 탭, 낡은 대장간 거점 제작 화면.

종료 조건: 재료 인벤토리 별도 구분 + 제작 레시피 충족/부족 표시 + 출처 3개 이상 연결 + 첫 제작 목표 3개 달성 가능 + 완제품 드랍과 제작 루트 공존. 누적 플레이 기준: 첫 제작 30~45분, 첫 희귀 장비 90~150분.

선행 의존성: M1·M2a·M2b·M3·M4 모두 완료. M4 시작 거점 더스트빌 4섹터 구조에 직접 얹는다.
