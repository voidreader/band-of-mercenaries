## 로드맵 요구사항 요약

M8.5 "재미 가시화와 폴리싱"은 M4~M8b에서 만든 기능을 플레이어가 재미로 인식하도록 가시화한다. 생활권, 세력, 전투, 제작, 위업, 간판 용병을 개별 화면에 흩어진 기능으로 두지 않고, "지금 무엇을 완성하고 있는가"와 "내 용병들이 어떤 이야기를 만들었는가"를 한눈에 보이게 한다.

핵심 목표는 이미 만든 시스템을 재미로 번역하는 것이다. 생활권 완성도 대시보드, 간판 용병 솔로/소수정예 루프, 전투 감정 반응, 히든 스탯, UI 폴리싱, 주간 기여도 랭킹 MVP를 한 묶음으로 검증한다.

선행 의존성: M8b 완료.

## 포함 범위

M8.5는 다음 범위를 포함한다.

- 생활권 완성도 대시보드.
- 간판 용병 1인 의뢰와 소수정예 의뢰.
- 동료 사망, 중상, 간판 용병 위기, 전멸 직전 상황의 감정 반응.
- 불굴, 투지, 운, 공포 저항, 전장 감각 등 히든 스탯 해금.
- 홈, 파견, 용병 상세, 이동, 제작 화면 UI 폴리싱 1차.
- 지역 썸네일, 세력 엠블럼, 아이템 아이콘, 용병 실루엣 또는 초상화, 엘리트 몬스터 이미지 1차 적용.
- 주간 더스트플레인 생활권 기여도 랭킹 MVP.
