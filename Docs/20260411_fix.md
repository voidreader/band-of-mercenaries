## 파견화면 
- 퀘스트 선택 후 아래에서 올라오는 '파견 인원 선택' UI가 화면의 5분의 4까지 올라올수있도록 수정 필요. 지금은 반절까지만 올라오고 있다. 
- 파견 인원 선택 UI에 선택한 퀘스트의 정보도 함께 표시 필요. 많이 올라와서 선택한 퀘스트 가리는 상황을 위해서 필요하다. 
- 사망한 용병이 '파견 인원 선택' 에 같이 표시되고 있다. (선택은 할수없는 상태) 사망한 용병은 표시하지 않도록 수정 필요.
- '파견 출발' 버튼이 안드로이드에서 하단 안드로이드 메뉴에 가려지는 현상. 안드로이드 메뉴보다 위에 위치하도록 수정 필요.
- iOS의 경우 하단 바에 가려지지 않는지 확인 필요하다.
- 위에 수정사항들을 작성했지만 파견 화면이 꽤 불편한 상태다. 파견 화면 자체를 좀 더 편리하게 개선하는 방안도 함께 고민해보면 좋을것 같다.

## 퀘스트 완료 팝업
- 퀘스트 완료 팝업에서 '확인' 버튼에 '확인'이라는 텍스트 대신 획득한 골드와 함께 '보상 수령'이라는 문구로 변경 필요.


## 홈 화면 - 최근활동 
최근활동이 많아졌을때, 스크롤이 되면 좋겠다. 10개까지 보여지고 그 이상은 스크롤로 볼 수 있도록 개선이 필요하다.
스크롤로 볼수 있는 최대 개수는 100개로 제한한다.
