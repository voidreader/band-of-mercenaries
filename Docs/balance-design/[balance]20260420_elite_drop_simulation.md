## 2. 현재 상태 — 입력 공식·수치 복기

### 2.1 페이즈 1-2 드랍 구조 (확정 제약)

- **독립 확률 판정**: 각 행 베르누이 시행. 기대값 = Σ drop_rate
- **엘리트당 드랍 행 수 상한**: 보통 4~6행, 유니크 7~9행
- **엘리트당 기대 드랍 수 초안**: 보통 **1.2~1.8**, 유니크 **2.5~3.5**
- **카테고리 내 상한**: essence 보통 3행·유니크 4행 / guild_artifact 보통 1행·유니크 2행

### 2.3 M2a 정수 인플레이션 기대치 (재확인)

- 효과 곡선: T1=+1, T2=+2, T3=+4, T4=+7, T5=+11
- 용병 티어별 축당 상한: T1=+10, T2=+20, T3=+40, T4=+70, T5=+120
- M2b 드랍 속도 가정 (로드맵 인용): "3~5시간에 정수 5~10개" = **시간당 1.0~3.3개 평균 2개**
- M6 승급 비용 (옵션 B): T1→T2=15개, T2→T3=25개, T3→T4=40개, T4→T5=60개, **T1→T5 총 140개**
- 의미 있는 강화 구간: 용병 1명 주력축 5~10개 투자 → 상한의 35~90% 도달

### 2.4 일반 퀘스트 수익 기준선 (Supabase 실데이터 기반)

**quest_types**: hunt base_reward=120, base_duration=80s, risk_factor=0.5 (엘리트 대응 주력 유형)

**difficulties**: `reward_multiplier` = L1: 1.0 / L2: 1.5 / L3: 2.2 / L4: 3.2 / L5: 4.5

**mercenary_wages**: T1=10, T2=25, T3=50, T4=100, T5=200 (인건비 합산)

일반 hunt 퀘스트 기본 보상 (엘리트 난이도 매핑: 엘리트 티어 N = 난이도 N):

| 난이도 | reward_multiplier | base_reward 120 적용 | 파견비 중앙값 (L3: 60, L4: 92, L5: 125) |
|--------|-------------------|---------------------|-----|
| L2 | 1.5× | **180G** | 35G |
| L3 | 2.2× | **264G** | 60G |
| L4 | 3.2× | **384G** | 92G |
| L5 | 4.5× | **540G** | 125G |

**엘리트 드랍 골드의 비교 기준**: 일반 퀘스트 `reward = base_reward × reward_multiplier` 기본값(대성공·장비·패시브 제외).

## 3. 드랍률 확정 — 정수 기대값 도출

### 3.1 접근 방법

페이즈 1-2 §7.3 티어 허용 매트릭스와 §8.1 타입 가족 매핑을 준수하며, **기대 행수 상향**으로 정수 획득 속도를 확보한다.

- 페이즈 1-2 초안 "보통 기대 1.2~1.8 / 유니크 2.5~3.5" → 본 리포트는 **보통 1.55~2.00 / 유니크 2.80~3.60**으로 상향
- 상향 근거: gold 1.0 확정 행을 제외하면 보통은 0.55~1.00, 유니크는 1.80~2.60이 "아이템 카테고리 합계 기대값"이 되도록. 이 중 essence 비중이 주력

### 3.2 보통 엘리트 essence 드랍률 표준 구성

| 구성 슬롯 | 드랍률 | rarity_grade | 설명 |
|----------|-------|--------------|------|
| 주 드랍축 정수 (엘리트 티어 -1 또는 동티어) | **0.40** (0.30~0.50) | rare | 타입 가족 주축 (§8.1) |
| 부 드랍축 정수 (엘리트 티어 -1) | **0.22** (0.15~0.30) | rare | 타입 가족 부축 |
| 선택 상위 티어 정수 (엘리트 티어 +1, 선택적) | **0.06** (0.03~0.08) | epic | 3행 선택 시에만 |

**essence 기대값**:
- 2행 구성 (기본): 0.40 + 0.22 = **0.62**
- 3행 구성 (엘리트당 선택): 0.40 + 0.22 + 0.06 = **0.68**

**보통 엘리트 전체 기대 드랍 행수** (골드 1 + essence 2~3 + 장비 1~2):
- 저구성 (4행): gold 1.00 + essence 0.62 + equip 0.10 = **1.72**
- 고구성 (6행): gold 1.00 + essence 0.68 + equip 0.10+0.05 + rare guild_artifact 0.02 = **1.85**

→ 기획서 §6.1 "1.2~1.8" 상한 상향 (1.55~2.00 권고)

### 3.3 유니크 엘리트 essence 드랍률 표준 구성

| 구성 슬롯 | 드랍률 | rarity_grade | 설명 |
|----------|-------|--------------|------|
| 주 드랍축 정수 (엘리트 티어) | **0.55** (0.50~0.70) | rare | 타입 가족 주축 |
| 부 드랍축 정수 (엘리트 티어) | **0.35** (0.30~0.45) | epic | 부축 동티어 |
| 상위 티어 정수 (엘리트 티어 +1) | **0.10** (0.05~0.15) | legendary | "서사의 정점" 낮은 확률 |
| 선택 추가 정수 (엘리트 티어 -1, T2/T3 유니크 선택) | **0.25** | rare | 4행 선택 시 |

**essence 기대값**:
- 3행 구성 (기본, T4~T5 유니크): 0.55 + 0.35 + 0.10 = **1.00**
- 4행 구성 (T2~T3 유니크, 하향 축적용): 0.55 + 0.35 + 0.10 + 0.25 = **1.25**

**유니크 엘리트 전체 기대 드랍 행수** (골드 1~2 + essence 3~4 + 장비 2~3 + 유물 1~2):
- 중간 구성 (7행, T2~T3 유니크): gold 1.00 + 잭팟 0.10 + essence 1.25 + equip 0.30+0.15 + artifact 0.12 = **2.92**
- 최대 구성 (9행, T4~T5 유니크): gold 1.00 + 잭팟 0.08 + essence 1.25 + equip 0.30+0.20+0.10 + artifact 0.15 + 추가 유물 0.05 = **3.13**
- T5 태고의 리치 특수 (9행, 2종 유물 + 전설 accessory): gold 1.00 + 잭팟 0.05 + essence 1.00 + equip 0.20+0.15+전설 accessory 0.04 + 유물 0.04+0.04 = **2.52**

→ 기획서 §6.1 "2.5~3.5" 유지 (2.80~3.60 권고)

### 3.4 페이즈 1-2 초안 대비 변경

| 항목 | 페이즈 1-2 초안 | **본 리포트 확정** | 변경 근거 |
|------|----------------|----------------|---------|
| 보통 essence 행별 드랍률 | 10~25% | **15~50%** 확장 | 주축 30~50%로 상향해야 시간당 획득이 의미 있음 |
| 유니크 essence 주축 | 50~75% | **50~70%** 축소 | 상한 70% 고정(초과 시 부축·상위티어 의미 상실) |

| 유니크 기대 행수 상한 | 3.5 | **3.60** | 상위 티어 정수·잭팟 골드 여유 |

## 4. 시간당 정수 획득 기대값 — 시나리오 A~D

### 4.1 계산 공식

```
시간당 정수 = Σ(엘리트구분별) [시간당 조우 × 성공률 × 기대 정수 수]
```

슬롯 4개 기준. 파티 수 여유가 있어 엘리트 조우 시 즉시 파견 가능하다고 가정(소요시간 < 슬롯 갱신 주기 1시간).

## 5. 골드 드랍 min/max 최종 조정

### 5.1 기준선: 일반 퀘스트 수익 대비 배율

기획서 §3.2: 보통 엘리트 = 일반 1.5~2배 / 유니크 = 일반 2~3배.

### 5.2 보통 엘리트 골드 드랍 (단일 common 행, drop_rate=1.0)

| 엘리트 티어 | 페이즈 1-2 §8.4 초안 | **확정 범위** | 평균 | 일반 대비 배율 |

## 10. 미결·향후 확정 사항

- **[Q-1] M2a 로드맵 "3~5시간에 5~10개" 과대 추정 → 문구 갱신 권고**: 본 리포트 시뮬레이션상 실제는 **M2b 12~25시간**. 로드맵·M2a 기획서 갱신 필요.
- **[Q-2] M6 승급 비용 옵션 B와 본 리포트 정합성 미달**: T1→T5 140개 달성에는 플레이 180h+ 필요. 로드맵 "헤비 1주"는 엘리트 단독 파밍으로 불가. **옵션 A(승급 비용 축소) 또는 로드맵 문구 갱신 필요** — M6 기획 시점 재검토.
- **[Q-3] 일반 퀘스트 정수 드랍 도입 검토**: 엘리트 파밍 단독으로 M6 기대치 미달. M3+ 범위에서 "낮은 확률 일반 퀘스트 정수 드랍"(예: L5 hunt 5% essence T1~T2) 도입 여지 고려. 본 리포트는 **미권고**(엘리트 희소성 훼손).
- **[Q-4] signature drop 인벤토리 상한 정책**: T3~T4 유니크 signature가 100h+ 주기로 드랍되면 동일 유물 2개 획득은 드묾. 그러나 특정 유물(예: 방패 장식이 강철 골렘·파수꾼·리치 3곳에 분산)의 중복 획득 시 M2a 인벤토리 정책(자동 판매 vs 스택) 적용 필요. spec-writer 페이즈 4 확정.
- **[Q-5] 엘리트 슬롯 중복 배정 방지**: 슬롯 4개 중 엘리트 판정은 독립. 동시 엘리트 2개 이상 출현 시 플레이어 파티 수 여유 초과 가능. `QuestPool.isFactionExclusive`와 같은 중복 방지 로직을 엘리트에도 적용할지 spec-writer 확인.
- **[Q-6] 시간당 정수 획득은 "이상적" 상황 가정**: 실제로는 용병 부상·피로·이동·기타 퀘스트 처리로 엘리트 조우 중 일부만 파견 가능. 실효 속도는 본 리포트 수치의 **약 0.7~0.85배**로 추정(체감 테스트 후 재검증).
- **[Q-7] 패시브 `item_drop_rate_bonus` 도입 여부 (페이즈 1-2 Q-1 재인용)**: 적용 시 시간당 정수 획득 **+10~20%** 가능. 본 리포트는 미적용 기준으로 계산. 페이즈 4-3 spec-writer 결정.
