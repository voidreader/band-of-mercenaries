## 현재 상태

### 용병단 장비 4종 컨셉 (입력 기획서 확정)

| # | 슬롯 | 이름 | 효과 축 (원안) | 권장 티어 |
|---|---|---|---|:---:|
| 1 | banner | 용병단의 깃발 | `reputation_gain_modifier` + `gold_reward_multiplier`(경미) | T3 |
| 2 | artifact | 황금 저울 | `gold_reward_multiplier` | T3 |
| 3 | artifact | 명예의 뿔피리 | `recruit_high_tier_chance` | T4 |
| 4 | artifact | 수호자의 방패 장식 | `injury_rate_modifier` 또는 `travel_event_bonus` | T5 |

### 기존 상한·중첩 체계 (M1 확정)

| 효과 타입 | 스태킹 | 상한 | 기존 소스 |
|---|---|---|---|
| `gold_reward_multiplier` | 가산 | **+0.80 공유** (`QuestCalculator.calculateReward` `passiveRewardBonus` clamp) | 세력(+0.12)·명성 누적(+0.10)·전설 ④ |
| `quest_success_rate_bonus` | 가산 | **+0.20 공유** (role/trait ±10%p 독립) | 세력·명성 |
| `recruit_high_tier_chance` | 가산 | 제한 없음 | 주점 Lv25 +0.15(T3~T5 분배) · 혈계 귀족회 +0.04(T4~T5) |
| `recovery_time_reduction` | 곱셈 | 하한 0.10 | 세력·명성·의무실 |

### 개인 장비 전설 유니크 효과 (M2a 페이즈 2 산출물 1 확정)

| 카테고리 | 소속 | 해당 키 |
|---|---|---|
| ③ `damage_resistance` | 전설 개인 장비 | `injury_rate_modifier` / `death_rate_modifier` |
| ④ `reward_bonus` | 전설 개인 장비 | `gold_reward_multiplier` (+0.80 공유 상한) |

**관찰**: 용병단 장비가 건드리는 `gold_reward_multiplier`·`injury_rate_modifier`는 **개인 장비 전설 유니크 효과와 같은 키**를 공유한다 → 중첩 정책이 교차 오염되지 않도록 정합성 확인 필요.

### 3슬롯 + 4종 경쟁 구조 (Q1 확정)

- banner 슬롯(1): 깃발 고정
- artifact 슬롯(2): 저울/뿔피리/방패 **3종 중 2종 경쟁**
- 플레이어는 banner 1 + artifact 2 = **3종 동시 가동**

**동시 가동 가능한 3가지 조합:**

| 조합 | 특성 |
|---|---|
| 깃발 + 저울 + 뿔피리 | 경제+모집 특화 |
| 깃발 + 저울 + 방패 | 경제+안전 특화 |
| 깃발 + 뿔피리 + 방패 | 모집+생존 특화 |

개인 장비 accessory 2와 동일한 "3택 2" 패턴으로 **전술 선택 레이어**를 형성.

## 데이터 분석

### 분석 1 — 스케일 철학 확정 (Q2)

**선택**: **A 경미** (체감 보조 레이어).

| 옵션 | 각 지표 아이템당 강도 | 포지션 |
|:---:|---|---|
| **A 경미 (확정)** | gold +3~7% / recruit +1~3%p / injury -3~7% / reputation +5~12% | 세력·명성보다 한 단계 낮은 약한 레이어 |
| B 세력급 | gold +5~12% / recruit +2~3%p / injury -5~12% / reputation +10~15% | 세력 1개 가입 수준 |
| C 강한 | gold +8~15% / recruit +3~5%p / injury -8~15% / reputation +15~25% | 개인 장비와 경쟁 |

**근거**:
- 용병단 장비는 **중립 기본 장비** 포지션(initial_item_set 기획서). M4 세력 전용 장비가 진짜 주력이 되는 **확장 훅을 위해 기본 장비는 경미**하게 유지
- M2a는 인프라 검증 마일스톤 — 효과 인플레이션 통제 우선
- 개인 장비 풀세트(+27% partyPower) + 세력 3개 가입(+27% 보상 등) 대비, 용병단 장비는 "있으면 좋은 보조"로 체감되는 무게감이 레이어 간 균형

### 분석 2 — 티어 곡선 형태

**선택**: **완만 가속** (개인 장비 B 곡선과 톤 통일).

| 티어 | 배율 | 근거 |
|:---:|:---:|---|
| T3 | ×1.0 (기본) | 컨셉 권장 분포 T3×2의 기준선 |
| T4 | ×1.5 | 개인 장비 B 곡선 T4/T3 = 10/6 ≈ 1.67과 근접, 완만 |
| T5 | ×2.0 | T5/T3 = 15/6 = 2.5 대비 보수적 (A 스케일 유지) |

개인 장비 B 곡선 T5/T3 = 2.5이지만, 용병단 장비는 A 스케일 철학상 **T5/T3 = 2.0으로 억제**하여 후속 마일스톤 세력 전용 T5 장비 확장 여유 확보.

### 분석 3 — 효과 축별 기준 수치 매트릭스 (A 스케일)

| `recruit_high_tier_chance` | +0.01 | **+0.02** | +0.03 | T4 기준점 |
| `injury_rate_modifier` | -0.03 | -0.05 | **-0.07** | 곱셈 스태킹 전제 |
| `reputation_gain_modifier` | **+0.05** | +0.08 | +0.12 | 신규 카탈로그, 상한 +0.30 선제 |

### 분석 4 — 4종 아이템별 수치 확정

| # | 슬롯 | 이름 | 티어 | `effect_json` | 비고 |
|:-:|---|---|:---:|---|---|
| 1 | banner | 용병단의 깃발 | T3 | `{"reputation_gain_modifier": 0.05, "gold_reward_multiplier": 0.02}` | **복합 효과 (A 스케일 내 예외)**. gold는 "경미" 표현을 위해 T3 기본값(+0.03)보다 낮은 +0.02 |
| 2 | artifact | 황금 저울 | T3 | `{"gold_reward_multiplier": 0.03}` | 경제 특화 |
| 3 | artifact | 명예의 뿔피리 | T4 | `{"recruit_high_tier_chance": 0.02}` | 모집 특화 |
| 4 | artifact | 수호자의 방패 장식 | T5 | `{"injury_rate_modifier": -0.07}` | 안전 특화. **일반 T5 (전설 규격 미적용)** |

**수호자의 방패 장식 효과 선택**: 컨셉 기획서의 `injury_rate_modifier` 또는 `travel_event_bonus` 중 **`injury_rate_modifier` 단일 선택**. 근거:
- `travel_event_bonus`는 M2a에 신규 카탈로그 추가 필요 + travel 결과 판정 로직 확장 부담
- `injury_rate_modifier`는 전설 개인 장비 ③과 경로 공유 (spec 단일 구현)
- A 스케일 단순화

### 분석 5 — 전설 규격 미적용 (Q3)

**선택**: **C 보류** (M2a 일반 T5만, 전설 규격은 M2b/M3/M4 세력 전용 확장 시 도입).

**근거**:
- initial_item_set: 용병단 장비 4종 = "중립 기본 장비" 포지션. 전설은 세력 전용 확장에 양도
- 개인 장비 전설 유니크 효과 5 카테고리 구현만으로 spec 부담 충분
- M2a 인프라 검증에서는 규격 단순화 우선

### 분석 6 — 개별 축 인플레이션 검증

#### 6-A. `gold_reward_multiplier` 종합 중첩 (+0.80 공유 상한)

**최악 케이스 (explore, 모험가 길드, 명성 A, 깃발+저울, 전설 개인 장비 ④ 광란검):**

| 소스 | 값 |
|---|:---:|
| 세력 모험가 길드 explore | +0.12 |
| 명성 D+B 누적 all | +0.10 |
| 전설 개인 장비 ④ `reward_bonus` | +0.12 |
| 용병단 장비 깃발 (경미 gold) | +0.02 |
| 용병단 장비 황금 저울 | +0.03 |
| **합산** | **+0.39** |
| **+0.80 공유 상한 clamp** | **+0.39 (여유 +0.41)** |

**판정**: 최악 케이스도 상한의 절반 이하. 인플레이션 위험 없음. **기존 `passiveRewardBonus` 가산 로직 그대로 흡수 가능**.

#### 6-B. `recruit_high_tier_chance` 과점 리스크

**최악 케이스: 주점 Lv25 + 혈계 귀족회 + 뿔피리 동시 활성:**

| 소스 | T4 기여 | T5 기여 |
|---|:---:|:---:|
| base 모집 확률 | 8% | 2% |
| 주점 Lv25 분배 (T3~T5 균등 추정) | +~2.5%p | +~2.5%p |
| 혈계 귀족회 T4~T5 | +~2%p | +~2%p |
| 뿔피리 T4 아이템 (균등 분배 가정) | +~1%p | +~1%p |
| **실효** | **~13.5%p** | **~7.5%p** |
| **T4+T5 합** | | **~21%p** |

**비교**: 무용병단 장비 (주점+혈계만) T4+T5 = ~20%p → **뿔피리 추가로 +1%p 증분**.

**판정**: 과점 심화 미미. 뿔피리가 전술 선택 레이어(artifact 3택2)에 참여하므로 **장착 기회비용**도 존재. 수용 가능.

#### 6-C. `injury_rate_modifier` 곱셈 스태킹

**최악 케이스: 방패 장식 + 전설 개인 장비 ③ 이그드라실 수피갑 (-0.15) + 야전병원 Lv25 (향후 도입 추정):**

| 방식 | 계산 | 결과 |
|---|---|:---:|
| 곱셈 (확정) | (1 - 0.07) × (1 - 0.15) = 0.93 × 0.85 | **0.7905 → -20.95%** |
| 하한 0.10 클램프 | max(0.7905, 0.10) | 통과 |

**실제 부상률 변화 (난이도 3, base 30%):**
- 무장비: 30% → 10명 파견 시 3명 부상
- 방패만: 30% × 0.93 = 27.9% → 약 2.8명
- 방패+이그드라실: 30% × 0.7905 = 23.7% → 약 2.4명

**판정**: 방패 단독은 체감 미미, 전설 개인 장비와 조합 시 유의미한 생존률 개선. 곱셈 스태킹으로 음수 위험 차단.

#### 6-D. `reputation_gain_modifier` 신규 상한 +0.30

**깃발 단일 소스로는 +0.05만 적용.** 상한 +0.30은 M3/M4에서 세력 전용 깃발(예: "모험가 길드의 깃발 +0.08") 추가 시를 대비한 선제 방어.

**시뮬레이션 (난이도 3, 성공 시 base 명성 30, 깃발 +5%):**
- 무장비: 30
- 깃발: 30 × 1.05 = **31.5** → 반올림 **32**

**판정**: 체감 미미하되, 장기 누적 시 "앱 켜둔 시간 보상" 감각. 깃발 장착 동기 제공.

### 분석 7 — 조합별 체감 시뮬레이션

**시나리오: D 등급 플레이어, T3 용병 2명, raid 난이도 2, 파견비 35G, 인건비 T3 100G.**

**기본 보상**: 100 × 1.5 = 150G, 순수익(성공) = 150 - 135 = 15G

| 용병단 장비 조합 | gold 추가 | 보상 | 순수익 | 증분 |
|---|:---:|:---:|:---:|:---:|
| 없음 | — | 150 | 15 | — |
| 깃발 + 저울 + 뿔피리 | +0.05 | 157.5 | 22.5 | **+50%** |
| 깃발 + 저울 + 방패 | +0.05 | 157.5 | 22.5 | **+50%** |
| 깃발 + 뿔피리 + 방패 | +0.02 | 153.0 | 18.0 | +20% |

**명성 기여 (성공, 난이도 2):**
- 기본 20
- 깃발 포함 조합: 20 × 1.05 = **21** (체감 미미)

**부상률 기여 (난이도 2 base 20%):**
- 방패 미포함: 20%
- 방패 포함: 20% × 0.93 = **18.6%**

**모집 기여 (유료 모집 1회, T4+T5 기본 10%):**
- 뿔피리 미포함: 10%
- 뿔피리 포함: 10% + 2 = **12%** (+20% 상대 증분)

**판정**: 순수익 +20~50% 증분은 A 스케일 철학 "경미"보다 크게 느껴질 수 있으나, **파견비·인건비 고정 차감 때문에 순수익 %로 증폭되는 것이 자연스러움**. 세력 패시브 +12% 순수익 증폭과 동일 메커니즘 → 수용.

## 문제점

### 중요 (해결됨, 문서화 필요)

**C1. 깃발의 복합 효과(`reputation_gain_modifier` + `gold_reward_multiplier`)는 M2a 4종 중 유일한 예외**
- 근거: 다른 3종은 단일 효과 축. 깃발만 복합
- 해결: `effect_json` 스키마는 **복합 키 허용**으로 유지. 단 분석 6-A의 +0.80 상한과 정합. 페이즈 4 명세에서 `ItemEffectService`가 깃발의 모든 키를 순회하여 각 상한에 가산하도록 구현

**C2. 개인 장비 전설 ③·④와 용병단 장비의 `injury_rate_modifier`·`gold_reward_multiplier` 키 공유 → 계산 경로 일원화 필요**
- 근거: 둘 다 `QuestCalculator.calculateReward`·`TraitEffectService.calculateInjuryRateModifier` 경로로 진입
- 해결: `PassiveBonusService`가 세력·명성·개인 장비·용병단 장비 모든 소스를 일괄 수집하여 동일 스태킹 규칙 적용. 페이즈 4 명세 필수

### 경미 (유지)

- 뿔피리 T4+T5 과점 리스크: +1%p 증분으로 수용 가능
- 깃발 명성 +5% 체감 미미: 깃발 장착 동기 제공을 위한 최소치로 유지
- 방패 단독 부상률 -7% 체감 미미: 전설 개인 장비 ③과 조합 전제

## 플레이어 체감 분석

### 초반 (F~E 등급, T2~T3 용병, 수동 지급 검증)

- 용병단 장비 1종 수령 시점에 "용병단 전체가 강해진다" 체감 → 개인 장비와 다른 감각
- 깃발(T3) 첫 수령 시 **순수익 20~50% 증분**이 체감 핵심. 명성 +5%는 부가
- A 경미 스케일이지만 파견비·인건비 고정 차감 때문에 **체감 강도는 세력 가입 수준**

### 중반 (D~C 등급, T3~T4 용병, 3종 동시 활성)

- artifact 3택2 선택지가 체감 발생 — "경제 vs 모집 vs 안전" 중 현재 성장 병목에 맞춰 교체
- 개인 장비 풀세트(+27% partyPower) + 세력 패시브(+12% 보상) + 용병단 장비(+5% gold, -7% injury)가 **다차원 누적**으로 누적 보상 지수적 상승

### 엔드게임 (B~A 등급, T4~T5 파티)

- 방패 장식의 injury -7% + 전설 멸혼결(-10%) 조합은 "수수하지만 의미 있는 엔드게임 옵션"

### 체감 리스크

- **교체 UX**: artifact 3택2 구조에서 "어떤 것을 뺄지" 선택 부담. 장착 화면에서 **현재 활성 효과 요약 + 교체 시 변동 프리뷰** 표시 필요 (페이즈 4 UX)
- **깃발 복합 효과 인지**: 유일한 복합 효과 아이템이므로 UI에서 `reputation_gain` + `gold_reward` 두 줄 분해 표시
- **용병단 장비 = 파티 공유** 개념이 초반 유저에게 직관적이지 않을 수 있음 → 튜토리얼 or 툴팁 안내

## 시뮬레이션 (조정안 적용 후)

### 시나리오 A — D등급 플레이어, 경제 특화 (깃발+저울+뿔피리)

**조건**: D등급, T3 용병 2명, explore 난이도 3, 세력 미가입.

| 항목 | 무장비 | 깃발+저울+뿔피리 |
|---|:---:|:---:|
| 기본 보상 | 80 × 2.2 = 176 | 동일 |
| `gold_reward_multiplier` 합산 | 0 | +0.05 (깃발 +0.02 + 저울 +0.03) |
| 보상 | 176 | 176 × 1.05 = **185** |
| 순수익 (인건비 100 + 파견비 60) | 16 | **25 G (+56%)** |
| 명성 획득 (성공, 난이도 3) | 30 | 30 × 1.05 = **32** |

**판정**: 순수익 +56%는 A 스케일 치고 큰 증분이나, 파견비·인건비 고정 차감 때문에 자연스러운 증폭. D등급 플레이어의 성장 가속에 적합.

### 시나리오 B — C등급 플레이어, 생존 특화 (깃발+뿔피리+방패)

**조건**: C등급 + 전설 개인 장비 이그드라실 수피갑 보유, T4 용병 3명, raid 난이도 4 (base 부상률 45%).

| 항목 | 방패 미장착 | 방패 장착 |
|---|:---:|:---:|
| `injury_rate_modifier` 합산 (곱셈) | -0.15 (이그드라실 단독) | (1-0.15)×(1-0.07) = 0.7905 |
| 실제 부상률 | 45% × 0.85 = **38.25%** | 45% × 0.7905 = **35.57%** |
| 10명 파견 시 부상 인원 기대 | 3.8명 | **3.6명** |
| 모집 확률 T4+T5 (뿔피리 효과) | 기본 10% | 기본 10% + 2 = **12%** |

**판정**: 부상률 -2.7%p 추가 감소는 체감 미미 (엔드게임 완충 역할). 뿔피리 +2%p는 유료 모집 시 체감.

### 시나리오 C — `gold_reward_multiplier` 상한 테스트 (엔드게임 풀스택)

**조건**: A등급, 모험가 길드 가입, T5 용병 3명 explore 난이도 5, 전설 개인 장비 ④ 보유, 깃발+저울 장착.

| 소스 | 값 |
|---|:---:|
| 세력 모험가 길드 explore | +0.12 |
| 명성 누적 D+B all | +0.10 |
| 전설 개인 장비 ④ `reward_bonus` | +0.12 |
| 용병단 장비 깃발 gold | +0.02 |
| 용병단 장비 저울 | +0.03 |
| **합산** | **+0.39** |
| +0.80 공유 상한 clamp | **+0.39 (여유 +0.41)** |

| 항목 | 값 |
|---|:---:|
| 기본 보상 | 80 × 4.5 = 360 |
| 보상 | 360 × 1.39 = **500** |
| 인건비 (T5 ×3) | 600 |
| 파견비 (난이도 5 평균) | 125 |
| 순수익 (성공) | **−225 G** |

**판정**: 난이도 5는 대성공(×2 보상) 전제가 아니면 손실. 실제 플레이는 대성공 비중 높은 파티 구성(트레잇·role synergy) → 기대값 +250G 수준. 상한 +0.80은 결코 도달하지 않음을 확인.

### 시나리오 D — artifact 3택2 전술 선택 체감

**시나리오**: C등급 플레이어가 현재 어떤 2개를 장착할지 고민.

| 현재 상황 | 권장 조합 | 이유 |
|---|---|---|
| 골드 쌓이는 중, 시설 건설 대기 | 저울 + 뿔피리 | 경제 특화 (지금은 gold가 성장 병목) |

| 새 난이도 도전 (부상 누적 우려) | 저울 + 방패 | 골드 + 부상률 감소 |

**판정**: 3택2 구조가 **상황별 최적해가 다름** → 전략 깊이 형성. A 스케일로 수치가 작아도 선택 자체는 의미 있음.
