# 아키텍처 구현 노트

이 문서는 `CLAUDE.md`의 "게임 핵심 시스템 로직" 섹션에서 분리한 **시스템별 상세 구현 노트**다.
CLAUDE.md에는 각 시스템의 진입점과 비자명 규칙만 요약되어 있으며, 메서드 시그니처·HiveField 번호·단계별 trailing 로직·마일스톤 태그 등 세부 서술은 이 문서에서 관리한다.

> **주의**: 단일 진실 공급원(source of truth)은 항상 코드다. 이 문서는 구현 의도와 단계별 흐름의 참고 자료이며, 코드 변경 시 함께 갱신하되 신규 세션 컨텍스트에 매번 로드되지는 않는다. typeId 점유·HiveField 다음 번호 표는 CLAUDE.md "영속성(Hive)" 섹션에 유지된다.
