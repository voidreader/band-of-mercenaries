### 간판 용병 솔로/소수정예 의뢰 시스템 추가 (M8.5 페이즈 4 #2)

- 솔로(1인) 3종 + 소수정예(페어 2인·삼인행 3인) 2종 의뢰가 NamedTier에 등장. 의뢰 카드에 ⭐/⭐⭐/⭐⭐⭐ 배지로 시각 차별화.
- 솔로 의뢰는 사망 저항 cap 0.95, 소수정예는 0.90로 보호. 체인 주인공이 솔로 의뢰를 수행하는 경우 솔로 cap(0.95)이 우선 적용.
- 의뢰 인원이 정확히 강제됨 — 솔로는 1명, 페어는 2명, 삼인행은 3명만 파견 가능. 솔로는 다른 용병 선택 시 자동 교체.
- 보상 배수 솔로 ×1.8~2.2 골드 / ×1.7~2.0 명성, 소수정예 ×1.4~1.6 / ×1.3~1.5. 가중치 α=2(M6 +3보다 희소), 쿨다운 48h/36h.
- 4 신규 칭호(외로운 늑대 / 은빛 페어 / 삼인행의 일원 / 굽히지 않은 자) — 솔로/페어/삼인행 완수 횟수 누적으로 자동 발급.
- 솔로 의뢰 성공 시 M8a 아티팩트/장신구 보장 드랍 + 신규 2 아티팩트 확률 드랍. 이미 보유 시 골드(100G × difficulty)로 자동 변환.
- 솔로 #1 성공 시 후속 메시지 활동 로그 노출("외로운 늑대의 이름이 또 한 번 입에 오르내렸다").
- 솔로/소수정예 의뢰 실패로 부상 시 신규 활동 로그 메시지 발급(`'<prefix> 의뢰 "{이름}" — {용병}이(가) 중상으로 귀환했다'`).
- 의뢰 상세 화면 보상 미리보기에 지명 배수가 정확히 반영되어 실제 보상과 화면 정합.
