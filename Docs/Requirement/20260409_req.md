
# 요구사항 문서

## 밸런스
- 파견 비용이 너무 비싸다. 소요시간 2분내 파견도 100G가 들어가는데, 많이 하향해야하며, 소요시간이 길어질수록 비용이 증가하는 구조로 변경해야한다. 최대 비용은 200G로 제안하는것을 제안한다. 


## 파견 
- 첫 실행시 퀘스트 생성 버튼을 눌러야 퀘스트가 생성되는데, 시작부터 퀘스트가 생성되어 있어야한다.
- 활성화된 퀘스트는 1시간마다 새로운 퀘스트로 갱신되어야한다.
- 활성 퀘스트가 5개 미만인 경우 퀘스트 채우기 버튼을 통해서 퀘스트를 생성할 수 있어야한다. 
- 파견할 퀘스트를 선택할때 화면 하단에서 파견인원 선택이 뜨는데 매번 스크롤을 내려야해서 불편하다. 아래서 올라오는 리스트 UI로 변경해서 파견 인원을 선택할 수 있도록 하자. 


## 홈 화면 
- 가운데 공간이 너무 비어보인다. 좋은 아이디어 제안이 없는지?? 
- 진행 상황이 종료된 경우 바로 결과 팝업이 뜨지 않고 '파견' 으로 들어가야 보여진다. 홈 화면에서 볼 수 있으면 좋겠다. 
  

## 이동
- 파견된 용병이 있는 경우 이동할 수 없어야한다. 
  - 이동 버튼 비활성화 및 버튼 텍스트를 '파견된 용병이 있습니다' 라고 변경해서 보여주기 

## 모집
- 방출 기능 필요하다.
  - 레벨, 티어에 비례해서 퇴직금을 지급하고 방출할 수 있게 하자.
  - 방출된 용병은 다시 모집할 수 없다. 

## 방치형 기능
- 방치형 기능이 필요하다. 미접속했다가 접속했을때 접속시간에 따라 소량의 골드를 지급할 수 있도록하자. 파견시 골드가 필요하기 때문에 골드를 벌 수 있는 수단 필요 


## 기타 이슈 

### 퀘스트 완료 화면
- 퀘스트 완료 화면을 띄워놓고 있으면 화면이 계속 깜빡깜빡하는 현상이 있다.
- 퀘스트 완료 화면을 계속 열어놓고 있으면 어느순간 '확인'버튼이 더이상 눌리지않고 팝업도 닫을수 없음.

### 설정
'시간 가속 모드'가 제대로 동작하지 않음. 10배, 100배 둘다 동작하지 않음
