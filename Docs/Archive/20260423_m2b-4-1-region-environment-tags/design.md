### 핵심 문제: 폐허 밀집 구조

ruins-테마 region_name 4종 합산:

| region_name | 수 | 비율 |
|------------|----|----|
| 폐허 | 30 | |
| 전쟁터 | 20 | |
| 고대유적 | 18 | |
| 마계경계 | 10 | |
| **합계** | **78** | **39.2%** |

세계 리전의 40%가 ruins-테마 → ruins 단순 합계 78개는 목표 상한 39의 **2배**. coast/swamp는 해당 이름의 region_name이 아예 없어 **0**개.

### 달성 불가능한 최소값

세만틱 정합성을 유지할 때 ruins의 물리적 하한:

| 제거 불가 이유 | ruins 기여 |
|-------------|---------|
| 폐허(30): "폐허" = ruins 그 자체 | 30 |
| 고대유적(18): "고대유적" = ancient ruins | 18 |
| **최소 ruins** | **48** |

전쟁터·마계경계에서 ruins를 모두 제거해도 **ruins ≥ 48** — 원 목표 39 달성 불가.

## 문제점

### P-1. ruins 과다 (목표 21~39 → 실제 48~78)
세계관이 폐허·전쟁 서사 중심으로 설계되어 있어 구조적 과다. 조정 없이는 ruins 관련 엘리트 몬스터가 다른 태그 대비 **2배 빈번하게** 출현한다.

### P-2. coast/swamp 전무 (목표 각 18~33/11~20 → 실제 0)
해안/습지 이름의 region_name이 없으므로, 보조 태그 부여 방식만으로 목표치를 달성해야 한다.

### P-3. underground 초과 (목표 14~26 → 실제 34)
고대유적·마계경계·심연이 모두 underground-테마 → T6~T10 지역의 지하 엘리트 집중 발생.

## 조정 제안

### 핵심 전략

1. **전쟁터(T5)**: ruins 제거 → [plains] only. 현재 진행 중인 전장은 폐허가 아닌 활성 전투 지역. ruins 기여 -20
2. **마계경계(T8)**: ruins·underground 제거 → [mountain] only. 마계 진입을 막는 산악 방벽으로 재해석. ruins -10, underground -10
3. **고대유적(T6) 절반**: ruins 제거 → [underground] only. 18개 중 9개는 지상 유적 없이 지하만 남은 고대 도시/미로 해석. ruins -9
4. **초원(T1) 15개**: [plains,coast] 보조 태그 추가. 최저 티어 개활지 중 해안가 초원. coast +15
5. **숲(T2) 13개**: [forest,swamp] 보조 태그 추가. 습한 원시림/안개 밀림. swamp +13
6. **황무지(T7) 3개**: [desert,coast] 보조 태그 추가. 해안 건조지대. coast +3

## 분기 ID 배정 규칙 (data-generator 지침)

동일 region_name 내에서 보조 태그 부여 대상을 결정하는 방법:

| region_name | 분기 수량 | 권장 배정 기준 |
|------------|---------|-------------|
| 초원(40) | coast 15개 | region_id 오름차순 정렬 후 **상위 15개** (세계 끝자락 = 해안가) |
| 숲(35) | swamp 13개 | region_id 오름차순 정렬 후 **중간 13개** (ID 12번째~24번째) |
| 고대유적(18) | ruins+un 9개 | region_id 오름차순 정렬 후 **하위 9개** (낮은 ID = 더 오래된 유적) |
| 황무지(15) | coast 3개 | region_id 오름차순 정렬 후 **하위 3개** |

ID 기준이 자의적으로 느껴지면, data-generator가 **균등 배분**(N개마다 1개)으로 대체 가능.

## data-generator 수치 가이드

- **대상 타입**: `region-environment-tag`
- **대상 테이블**: `regions` (UPDATE, 199행)
- **허용 태그**: `["ruins","forest","swamp","mountain","desert","coast","underground","plains"]`
- **분기 규칙**:

```
초원(40):   ID ASC 기준 마지막 15개 → ["plains","coast"] / 나머지 → ["plains"]
숲(35):     ID ASC 기준 중간 13개 → ["forest","swamp"] / 나머지 → ["forest"]
폐허(30):   전부 → ["ruins"]
산악(25):   전부 → ["mountain"]
전쟁터(20): 전부 → ["plains"]
고대유적(18):ID ASC 기준 첫 9개 → ["ruins","underground"] / 나머지 → ["underground"]
황무지(15): ID ASC 기준 첫 3개 → ["desert","coast"] / 나머지 → ["desert"]
마계경계(10):전부 → ["mountain"]
심연(6):    전부 → ["underground"]
```

- **검증 목표 분포**:
  - plains 60 / forest 35 / mountain 35 / ruins 39 / coast 18 / desert 15 / underground 24 / swamp 13
- **balance 근거**: 위 분기 규칙 적용 시 8개 태그 전부 `types/region-environment-tag.md` 목표 범위(±30%) 내 달성
