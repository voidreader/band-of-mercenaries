## 현재 상태

M8a 세력 상점은 모험가 길드, 상인 연합, 전사 길드 3개 세력만 대상으로 한다. 각 세력은 상품 6개를 제공하며, M8a 전체 상점 상품 수는 18개다.

Supabase 실데이터 기준으로 현재 경제 기준선은 다음과 같다.

| 항목 | 현재 값 |
|------|---------|
| E 랭크 요구 명성 | 300 |
| D 랭크 요구 명성 | 2,000 |
| 용병 임금 | T1 10G / T2 25G / T3 50G / T4 100G / T5 200G |
| 주요 퀘스트 기본 보상 | explore 80G / escort 90G / raid 100G / hunt 120G |
| 난이도 보상 배수 | d1 1.0 / d2 1.5 / d3 2.2 / d4 3.2 / d5 4.5 |
| 기존 대표 세력 패시브 | 모험가 길드 explore 보상 +12%, 상인 연합 escort 보상 +12% + 방치 +10%, 전사 길드 raid/hunt 성공률 +5%p |
| M7 외래 좌판 기준 | Tier 3부터 3종, Tier 4부터 6종 + 20% 할인 |

M8a 상점은 골드 소모처를 추가하지만, 기존 제작 루프를 무력화해서는 안 된다. 따라서 세력 상점은 희귀 재료를 대량 판매하지 않고, "부족한 1~2개를 메우는 보조 경로"로 제한한다.

## 데이터 분석

### 상품 가격 기준선

기존 보상 공식상 난이도 2~4 성공 보상은 대략 다음 구간에 있다.

| 의뢰 | 난이도 | 총 보상 | T2 2명 임금 차감 후 순수익 예시 |
|------|--------|---------|-------------------------------|
| explore | 2 | 120G | 약 70G |
| escort | 2 | 135G | 약 85G |
| raid | 3 | 220G | T3 2명 기준 약 120G |
| hunt | 3 | 264G | T3 2명 기준 약 164G |
| escort | 4 | 288G | T3 3명 기준 약 138G |

M8a 상점 상품은 1회 의뢰 순수익 1~4회분 사이에 배치한다. 소모품·재료는 60~180G, 장비·보상형 상품은 260~700G를 기준으로 한다.

### 구매 시점 기준

M7 종료 직후 플레이어는 대체로 인프라 Tier 3~4, 랭크 D 전후, M7 레시피 2~4개를 보유한다. 이 시점에서 상점이 너무 비싸면 세력 접촉 체감이 끊기고, 너무 싸면 제작 재료 수급 곡선이 무너진다.

권장 구매 템포는 다음과 같다.

| 구간 | 플레이어 상태 | 구매 기대 |
|------|--------------|----------|
| 접촉 직후 | 평판 1~10, 가입 전 | 저가 소모품 또는 반응용 상품 1개 |
| 가입 직후 | 평판 11~30 | 세력 정체성 상품 1~2개 |
| 신뢰 진입 | 평판 31~60 | 장비·레시피·상징 보상 1개 |
| 고급 신뢰 | 평판 61 이상 | 세력당 1개 고가 상품만 |

## 문제점

### 1. 상인 연합 재료 판매가 제작 루프를 압도할 위험

상인 연합은 M7 region_exclusive 재료와 자연스럽게 연결된다. 하지만 유목민 가죽끈, 해초 약재, 들꽃 약초를 상시 판매하면 region 31·127 외출 동기가 약해진다.

따라서 상인 연합 재료 상품은 다음 제약을 둔다.

- 재료 묶음은 1일 1회 또는 갱신당 1회 제한 재고로 둔다.
- region_exclusive 재료는 단품 1~2개만 판매한다.
- 제작 목표 하나를 통째로 해결할 수 있는 수량은 판매하지 않는다.

### 2. 전사 길드 상품이 전투 성공률과 과도 중첩될 위험

전사 길드는 이미 raid/hunt 성공률 +5%p 패시브를 제공한다. 상점 장비가 추가 성공률을 직접 제공하면 성공률 95% 상한에 빠르게 닿는다.

전사 길드 상품은 STR/VIT 소폭 보정, 부상률 완화, 전투 보고서 태그 같은 보조 효과 중심으로 설계한다. 직접 성공률 보너스는 M8a 상점에서 제외한다.

### 3. 모험가 길드 상품이 기존 explore 보상 +12%와 중첩될 위험

모험가 길드는 이미 explore 보상 +12%를 제공한다. 상점 상품은 추가 골드 배수보다 탐험 의뢰 노출, 보고서 품질, 기록 보상 같은 비경제적 체감에 가깝게 둔다.

## 플레이어 체감 분석

M8a 상점은 "돈을 모아 대형 시설을 사는 경험"이 아니라 "세력이 내 사건을 알아보고 작은 물건을 건넨다"는 체감을 목표로 한다. 80~180G 상품은 접촉 직후 바로 구매 가능하고, 420~700G 상품은 세력 지명 의뢰 2~5회 이후 구매 가능하다.

이 구간은 M7 후반의 의뢰 순수익과 맞다. 플레이어는 한 번에 모든 상품을 사지 못하지만, 한 세력에 집중하면 1~2시간 안에 해당 세력의 핵심 상품 2~3개를 확보할 수 있다.

## 시뮬레이션

### 상인 연합 예시

가정은 다음과 같다.

- 접촉 시점: 인프라 Tier 3, 평판 1
- 가입 후 지명 의뢰 3회 성공
- 지명 의뢰 평균 순수익: 120~180G
- 일반 의뢰 병행 순수익: 200~300G

| 시점 | 누적 추가 골드 | 구매 가능 |
|------|---------------|----------|
| 접촉 직후 | 100~200G | 밀봉 계약서 또는 보급 식량 |
| 지명 의뢰 2회 후 | 350~500G | 유목민 가죽끈 묶음 + 해초 약재 묶음 |
| 지명 의뢰 4~5회 후 | 650~900G | 황금 저울 복제품 또는 교역 인장 조각 |

상점은 제작 루프를 대체하지 않고 부족분을 보정한다.

## data-generator 수치 가이드

- **대상 타입**: 신규 타입 스펙 필요. 후보명 `faction-shop-item`
- **대상 테이블**: 신규 후보 `faction_shop_items`, 필요 시 `items`
- **수치 범위**:
  - `price_gold`: 80~750
  - `min_reputation`: 1 / 11 / 31 / 61 중 하나
  - `stock_limit`: 1 또는 2
  - `restock_hours`: 제한 재고는 24
- **외래 키 제약**:
  - `faction_id`: `faction_adventurers_guild`, `faction_merchants_alliance`, `faction_warriors_guild`
  - `item_id`: 기존 `items.id` 또는 M8a 신규 item id
- **balance 근거**: 상품 가격은 M7 후반 1~5회 의뢰 순수익에 맞추며, region_exclusive 재료는 제작 목표를 한 번에 해결하지 않는 수량으로 제한한다.
