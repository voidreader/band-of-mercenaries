# 용병단 이야기 — 게임 소개 및 구현 현황

> 기획자, 아트 직군 등 프로젝트 참여자를 위한 가이드  
> 마지막 업데이트: 2026-04-13

---

## 이 게임은 어떤 게임인가요?

**용병단 이야기**는 당신이 용병단의 단장이 되어 용병들을 고용하고, 대륙 곳곳의 의뢰를 수행하며 명성을 쌓아가는 **방치형 텍스트 전략 시뮬레이션** 모바일 게임입니다.

직접 전투하지 않습니다. 대신 **어떤 용병을 보낼지, 어디로 이동할지, 어떤 의뢰를 받을지**를 결정합니다. 파견을 보내면 실제 시간이 흐르고, 나중에 결과를 확인하는 방식입니다.

**핵심 감성**: "내 용병단이 성장하고 있다"는 느낌. 레벨업, 명성 상승, 더 어려운 지역 개방.

---

## 기본 플레이 흐름

```
① 용병 모집   →   ② 지역 이동   →   ③ 의뢰 파견   →   ④ 시간 대기   →   ⑤ 결과 확인
     ↑                                                                          |
     └──────────────────────── 골드로 재모집, 반복 ─────────────────────────────┘
```

### 단계별 설명

**① 용병 모집**
- 무료 모집(2시간 쿨타임)과 골드 지불 모집 두 가지 방식
- 5등급 티어로 구분된 85종의 직업 중 랜덤 배정
- 희귀 등급일수록 강하지만, 그만큼 인건비도 비쌈
- 모집 시 **선천 트레잇** 1~3개를 랜덤으로 보유한 채 합류

**② 지역 이동**
- 대륙에는 199개 지역(리전)이 존재하며, 5단계 등급으로 나뉨
- 이동 거리에 따라 시간 소요 (거리 1칸 = 30초)
- 이동 중 랜덤 이벤트 발생 (골드 발견, 도적 습격, 날씨 이변 등 12종)
- 상위 등급 지역은 명성 조건 달성 후 해금

**③ 의뢰 파견**
- 현재 위치에서 자동 생성된 의뢰 목록(5~8개) 중 선택
- 원하는 용병을 골라 파견. 많이 보낼수록 성공률 상승
- 파견 전 비용(인건비 + 파견비)이 선차감됨

**④ 시간 대기**
- 의뢰 난이도에 따라 소요 시간 다름
- 앱을 꺼도 시간이 흐름. 오래 자리를 비우면 방치 보상(최대 480G)도 지급됨

**⑤ 결과 확인**
- **대성공**: 보상 2배
- **성공**: 기본 보상 획득, 경험치 및 명성 상승
- **실패**: 보상 없음, 용병 부상
- **대실패**: 사망 위험 증가

## 세계관 설정

### 지역 구조

대륙은 **199개의 리전**으로 구성되며, 각 리전은 10개의 섹터로 세분화됩니다. 리전은 위험도에 따라 5단계 등급으로 나뉩니다.

| 등급 | 분위기 | 해금 조건 |
|------|--------|----------|
| Tier 1 | 평화로운 초보 지역 | 처음부터 개방 |
| Tier 2 | 약간의 위험이 있는 지역 | 명성 500 (신출내기) |
| Tier 3 | 숙련된 용병이 필요한 지역 | 명성 2,000 (일반) |
| Tier 4 | 위험한 고난도 지역 | 명성 8,000 (숙련) |
| Tier 5 | 극한의 도전 지역 | 명성 25,000 (정예) |

### 명성과 랭크

의뢰를 성공시킬수록 용병단의 명성이 쌓입니다. 명성이 일정 수준에 달하면 랭크가 오르고 새로운 지역이 열립니다.

| 랭크 | 호칭 | 필요 명성 |
|------|------|----------|
| F | 무명 | 0 |
| E | 신출내기 | 500 |
| D | 일반 | 2,000 |
| C | 숙련 | 8,000 |
| B | 정예 | 25,000 |
| A | 전설 | 80,000 |

## 용병 시스템

### 등급 (티어)

용병의 등급이 높을수록 강력하지만, 모집 확률이 낮고 인건비가 비쌉니다.

| 등급 | 모집 확률 | 인건비(의뢰당) | 분위기 |
|------|----------|--------------|--------|
| Tier 1 | 45% | 10G | 신입 용병 |
| Tier 2 | 30% | 25G | 경력 용병 |
| Tier 3 | 15% | 50G | 베테랑 용병 |
| Tier 4 | 8% | 100G | 엘리트 용병 |
| Tier 5 | 2% | 200G | 전설적인 용병 |

### 직업 (85종)

5단계 등급에 걸쳐 총 85가지 직업이 존재합니다. 각 직업마다 고유한 ATK(공격), DEF(방어), HP(체력), Speed(속도) 스탯이 다릅니다.

### 트레잇 시스템 (106종)

용병은 **선천 트레잇**(최대 3개)과 **후천 트레잇**(최대 4개)을 보유할 수 있습니다. 트레잇은 파견 성공률·데미지 보정 등 실전에 영향을 주며, 성장하는 용병의 개성을 형성합니다.

**8개 카테고리**

| 카테고리 | 설명 | 트레잇 유형 |
|----------|------|------------|
| Physical | 체력·강인함 등 신체 특성 | 선천 |
| Background | 출신·경력 배경 | 선천 |
| Talent | 타고난 재능 | 선천 |
| CombatStyle | 전투 스타일 | 후천 |
| Survival | 생존 능력 | 후천 |
| Behavior | 행동 패턴 | 후천 |
| Mental | 정신력 | 후천 |
| Experience | 경험치 축적형 | 후천 |

**획득 및 진화**

- **자동 획득**: 퀘스트 완료 시 23개 행동 지표(파견 횟수, 성공률, 특정 퀘스트 유형 등)가 축적되면 조건을 충족한 트레잇이 자동 부여됩니다.
- **단일 진화**: 보유 후천 트레잇이 지표 조건을 충족하면 동일 카테고리의 상위 `evolved` 트레잇으로 교체됩니다.
- **조합 진화**: 서로 다른 카테고리의 트레잇 2개를 조합해 원본을 소멸시키고 강력한 결과 트레잇을 획득합니다.
- **충돌 관계**: 일부 트레잇은 함께 보유할 수 없습니다 (16쌍).
- **트레잇 히스토리**: 소멸된 트레잇은 기록되어 재획득이 방지됩니다.

### 용병 상태

| 상태 | 설명 | 지속 시간 |
|------|------|----------|
| 정상 | 모든 능력치 정상 | - |
| 피곤함 | 능력치 80%로 감소 | 5분 |
| 부상 | 의뢰 파견 불가 | 난이도 × 10분 |
| 사망 | 영구 제거 | - |

### 레벨 성장

용병은 의뢰를 통해 경험치를 쌓고 최대 5레벨까지 성장합니다.

| 레벨 | 필요 경험치 |
|------|-----------|
| 1 | 0 |
| 2 | 100 XP |
| 3 | 350 XP |
| 4 | 850 XP |
| 5 | 1,850 XP |

### 용병 방출

더 이상 필요 없는 용병은 방출할 수 있습니다. 방출 시 퇴직금(인건비 × 레벨)을 지급합니다. 방출된 용병은 재모집되지 않습니다.

## 의뢰 시스템

### 의뢰 유형 (4종)

| 유형 | 기본 보상 | 분위기 |
|------|----------|--------|
| 약탈 | 100G | 공격적인 임무 |
| 토벌 | 120G | 몬스터/세력 토벌 |
| 호위 | 90G | 상인/귀족 호위 |
| 탐험 | 80G | 미지의 지역 탐사 |

### 난이도 (5단계)

| 난이도 | 적 전투력 | 보상 배수 |
|--------|----------|----------|
| 1 | 10 | 1.0배 |
| 2 | 20 | 1.5배 |
| 3 | 40 | 2.5배 |
| 4 | 60 | 3.5배 |
| 5 | 80 | 4.5배 |

### 성공률 계산

```
기본 50% + (아군 전투력 / 적 전투력 - 1) × 50%
      + 특성 보너스 + 퀘스트 보정 - 거리 패널티 + 랜덤 편차
최소 5% ~ 최대 95%
```

## 경제 시스템

### 기본 수익 구조

```
순수익 = 기본 보상 × 난이도 배수 - 인건비 - 파견비용
```

- **파견비용**: 의뢰 난이도와 소요 시간에 비례해 증가
- **인건비**: 파견한 용병 수 × 티어별 단가
- 비용은 파견 시작 시 선차감됨 (실패해도 돌려받지 못함)

### 골드 획득 경로

| 경로 | 금액 |
|------|------|
| 의뢰 성공 | 유형 × 난이도 배수 |
| 여행 중 골드 발견 | +20~120G |
| 방치형 보상 | 분당 1G, 최대 480G |

### 골드 소비 경로

| 경로 | 금액 |
|------|------|
| 용병 유료 모집 | 가변 |
| 의뢰 파견비용 | 난이도/시간 비례 |
| 인건비 | 용병 수 × 티어 단가 |
| 시설 업그레이드 | 시설별 차등 |
| 용병 방출 퇴직금 | 인건비 × 레벨 |

야영지에 시설을 건설하고 업그레이드하면 용병단 전체의 능력이 향상됩니다.

| 시설 | 효과 | 최대 레벨 | 업그레이드 비용 (예) |
|------|------|----------|-------------------|
| 훈련소 | 경험치 +10%~+50% | 5 | 500 / 1,000 / 2,000 / 4,000 / 8,000G |
| 의무실 | 부상 회복 시간 -10%~-50% | 5 | - |
| 주둔지 | 용병 최대 인원 +2~+10 | 5 | - |
| 정보망 | 의뢰 추가 생성 +1~+3개 | 3 | - |

## 여행 이벤트

이동 중에 다양한 이벤트가 발생할 수 있습니다. 이동 거리가 길수록 이벤트 발생 확률이 높아집니다(최대 80%).

| 이벤트 유형 | 예시 |
|-----------|------|
| 발견 | 버려진 보물 발견, 골드 획득 |
| 도적 습격 | 골드 손실, 용병 피해 |
| 날씨 이변 | 이동 지연 |
| 행운 | 예상치 못한 이득 |
| 조우 | 이동 중 특수 상황 |

(총 12종 이벤트)

## 활동 로그

플레이어의 모든 주요 행동은 활동 로그에 기록됩니다. 최대 100개의 로그가 유지됩니다.

기록되는 사건:
- 의뢰 완료 결과 (성공/실패/대성공/대실패)
- 용병 상태 변화 (부상, 사망, 회복)
- 이동 완료
- 용병 모집
- 용병 방출
- 용병 레벨업
