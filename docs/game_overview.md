# 용병단 이야기 — 게임 소개 및 구현 현황

> 기획자, 아트 직군 등 프로젝트 참여자를 위한 가이드  
> 마지막 업데이트: 2026-04-26 (M3 완료 반영)

---

## 이 게임은 어떤 게임인가요?

**용병단 이야기**는 당신이 용병단의 단장이 되어 용병들을 고용하고, 대륙 곳곳의 의뢰를 수행하며 명성을 쌓아가는 **방치형 텍스트 전략 시뮬레이션** 모바일 게임입니다.

직접 전투하지 않습니다. 대신 **어떤 용병을 보낼지, 어디로 이동할지, 어떤 의뢰를 받을지**를 결정합니다. 파견을 보내면 실제 시간이 흐르고, 나중에 결과를 확인하는 방식입니다.

**핵심 감성**: "내 용병단이 성장하고 있다"는 느낌. 레벨업, 명성 상승, 더 어려운 지역 개방.

---

## 기본 플레이 흐름

```
① 용병 모집   →   ② 지역 이동   →   ③ 의뢰 파견   →   ④ 시간 대기   →   ⑤ 결과 확인
     ↑                                                                          |
     └──────────────────────── 골드로 재모집, 반복 ─────────────────────────────┘
```

### 단계별 설명

**① 용병 모집**
- 무료 모집(2시간 쿨타임)과 골드 지불 모집 두 가지 방식
- 5등급 티어로 구분된 85종의 직업 중 랜덤 배정
- 희귀 등급일수록 강하지만, 그만큼 인건비도 비쌈
- 모집 시 **선천 트레잇** 1~3개를 랜덤으로 보유한 채 합류

**② 지역 이동**
- 대륙에는 199개 지역(리전)이 존재하며, 5단계 등급으로 나뉨
- 이동 거리에 따라 시간 소요 (거리 1칸 = 30초)
- 이동 중 랜덤 이벤트 발생 (골드 발견, 도적 습격, 날씨 이변 등 12종)
- 이동 완료 시 확률적으로 **선택지 이벤트** 발동 ("회상" 팝업)
- 상위 등급 지역은 명성 조건 달성 후 해금

**③ 의뢰 파견**
- 현재 위치에서 자동 생성된 의뢰 목록(5~8개) 중 선택
- 원하는 용병을 골라 파견. 많이 보낼수록 성공률 상승
- 파견 전 비용(인건비 + 파견비)이 선차감됨
- **연계 퀘스트** 진행 중이면 파견 화면 최상단에 고정 표시

**④ 시간 대기**
- 의뢰 난이도에 따라 소요 시간 다름
- 앱을 꺼도 시간이 흐름. 오래 자리를 비우면 방치 보상(최대 480G)도 지급됨

**⑤ 결과 확인**
- **대성공**: 보상 2배
- **성공**: 기본 보상 획득, 경험치 및 명성 상승
- **실패**: 보상 없음, 용병 부상
- **대실패**: 사망 위험 증가
- 결과마다 **서사 텍스트**("용병 김철수가 적 대장과 일대일 결투를 벌여 승리했다")가 표시됨

## 세계관 설정

### 지역 구조

대륙은 **199개의 리전**으로 구성되며, 각 리전은 10개의 섹터로 세분화됩니다. 리전은 위험도에 따라 5단계 등급으로 나뉩니다.

| 등급 | 분위기 | 해금 조건 |
|------|--------|----------|
| Tier 1 | 평화로운 초보 지역 | 처음부터 개방 |
| Tier 2 | 약간의 위험이 있는 지역 | 명성 500 (신출내기) |
| Tier 3 | 숙련된 용병이 필요한 지역 | 명성 2,000 (일반) |
| Tier 4 | 위험한 고난도 지역 | 명성 8,000 (숙련) |
| Tier 5 | 극한의 도전 지역 | 명성 25,000 (정예) |

### 명성과 랭크

의뢰를 성공시킬수록 용병단의 명성이 쌓인다. 명성이 일정 수준에 달하면 랭크가 오르고 새로운 지역이 열린다. 랭크가 상승하면 추가 보너스(퀘스트 보상 증가, 모집 할인 등)도 함께 해금된다. 보너스는 F부터 현재 랭크까지 누적 적용된다.

| 랭크 | 호칭 | 필요 명성 |
|------|------|----------|
| F | 무명 | 0 |
| E | 신출내기 | 500 |
| D | 일반 | 2,000 |
| C | 숙련 | 8,000 |
| B | 정예 | 25,000 |
| A | 전설 | 80,000 |

## 용병 시스템

### 등급 (티어)

용병의 등급이 높을수록 강력하지만, 모집 확률이 낮고 인건비가 비쌉니다.

| 등급 | 모집 확률 | 인건비(의뢰당) | 분위기 |
|------|----------|--------------|--------|
| Tier 1 | 45% | 10G | 신입 용병 |
| Tier 2 | 30% | 25G | 경력 용병 |
| Tier 3 | 15% | 50G | 베테랑 용병 |
| Tier 4 | 8% | 100G | 엘리트 용병 |
| Tier 5 | 2% | 200G | 전설적인 용병 |

### 직업 (85종)

5단계 등급에 걸쳐 총 85가지 직업이 존재합니다. 각 직업마다 고유한 STR(공격력), INT(스킬), VIT(체력/방어), AGI(속도/회피) 스탯이 다릅니다.

### 트레잇 시스템 (109종)

용병은 **선천 트레잇**(최대 3개)과 **후천 트레잇**(최대 4개)을 보유할 수 있습니다. 트레잇은 파견 성공률·데미지 보정 등 실전에 영향을 주며, 성장하는 용병의 개성을 형성합니다.

**8개 카테고리**

| 카테고리 | 설명 | 트레잇 유형 |
|----------|------|------------|
| Physical | 체력·강인함 등 신체 특성 | 선천 |
| Background | 출신·경력 배경 | 선천 |
| Talent | 타고난 재능 | 선천 |
| CombatStyle | 전투 스타일 | 후천 |
| Survival | 생존 능력 | 후천 |
| Behavior | 행동 패턴 | 후천 |
| Mental | 정신력 | 후천 |
| Experience | 경험치 축적형 | 후천 |

**획득 및 진화**

- **자동 획득**: 퀘스트 완료 시 26개 행동 지표(파견 횟수, 성공률, 특정 퀘스트 유형, 시설 혜택 횟수 등)가 축적되면 조건을 충족한 트레잇이 자동 부여됩니다.
- **단일 진화**: 보유 후천 트레잇이 지표 조건을 충족하면 동일 카테고리의 상위 `evolved` 트레잇으로 교체됩니다.
- **조합 진화**: 서로 다른 카테고리의 트레잇 2개를 조합해 원본을 소멸시키고 강력한 결과 트레잇을 획득합니다.
- **충돌 관계**: 일부 트레잇은 함께 보유할 수 없습니다 (16쌍).
- **트레잇 히스토리**: 소멸 또는 삭제된 트레잇은 기록되어 재획득이 방지됩니다.
- **트레잇 삭제**: 후천 트레잇(acquired/evolved)을 골드로 삭제할 수 있습니다. acquired 200G, evolved 500G. 의무실 레벨 조건(acquired Lv2, evolved Lv4) 충족 필요. 선천 트레잇은 삭제 불가.
- **여행 이벤트 선천 트레잇**: 이동 중 빈 선천 슬롯을 보유한 용병에게 선천 트레잇이 부여되는 이벤트가 발생합니다 (Physical/Background/Talent 카테고리별 3종).

### 용병 상태

| 상태 | 설명 | 지속 시간 |
|------|------|----------|
| 정상 | 모든 능력치 정상 | - |
| 피곤함 | 능력치 80%로 감소 | 5분 |
| 부상 | 의뢰 파견 불가 | 난이도 × 10분 |
| 사망 | 영구 제거 | - |

### 레벨 성장

용병은 의뢰를 통해 경험치를 쌓고 최대 5레벨까지 성장합니다.

| 레벨 | 필요 경험치 |
|------|-----------|
| 1 | 0 |
| 2 | 100 XP |
| 3 | 350 XP |
| 4 | 850 XP |
| 5 | 1,850 XP |

### 용병 방출

더 이상 필요 없는 용병은 방출할 수 있습니다. 방출 시 퇴직금(인건비 × 레벨)을 지급합니다. 방출된 용병은 재모집되지 않습니다.

## 의뢰 시스템

### 의뢰 유형 (4종)

| 유형 | 기본 보상 | 분위기 |
|------|----------|--------|
| 약탈 | 100G | 공격적인 임무 |
| 토벌 | 120G | 몬스터/세력 토벌 |
| 호위 | 90G | 상인/귀족 호위 |
| 탐험 | 80G | 미지의 지역 탐사 |

### 난이도 (5단계)

| 난이도 | 적 전투력 | 보상 배수 |
|--------|----------|----------|
| 1 | 10 | 1.0배 |
| 2 | 20 | 1.5배 |
| 3 | 40 | 2.5배 |
| 4 | 60 | 3.5배 |
| 5 | 80 | 4.5배 |

### 성공률 계산

```
기본 50% + (아군 전투력 / 적 전투력 - 1) × 50%
      + 파견 상성 보정 (±10%p 독립 상한)
      + 트레잇 보정 (±10%p 독립 상한)
      + 세력 패시브·랭크 보너스 (최대 +20%p 공유 상한)
      - 거리 패널티 + 랜덤 편차
최소 5% ~ 최대 95%
```

**파견 상성 시스템:** 직업군(warrior / specialist / mage / support / ranger / rogue)과 퀘스트 유형(약탈/토벌/호위/탐험)에 따라 성공률 보정이 달라진다. 파견 화면에서 추천 직업군 힌트와 유리한 용병 강조 표시를 제공하며, `?` 버튼으로 성공률 구성 요소를 레이어별로 확인할 수 있다.

### 엘리트 몬스터 (M2b)

일부 의뢰에는 특수 강적인 **엘리트 몬스터**가 등장한다. 총 39종(보통 31종 + 유니크 8종) 두 계층으로 구분되며, 리전 등급 범위와 환경 태그(숲/던전 등)에 맞게 확률적으로 배정된다.

| 계층 | 표시 | 설명 | 드랍 |
|------|------|------|------|
| 보통 엘리트 | 🔥 | 일반보다 강한 특수 강적 | 보너스 골드 |
| 유니크 엘리트 | ★ | 고유 이름과 설명을 가진 전설적 강적 | 보너스 골드 + 아이템 |

파견 화면에서 엘리트 퀘스트 카드는 별도 색상 강조와 배지(🔥 / ★)로 구분된다.

## 경제 시스템

### 기본 수익 구조

```
순수익 = 기본 보상 × 난이도 배수 - 인건비 - 파견비용
```

- **파견비용**: 의뢰 난이도와 소요 시간에 비례해 증가
- **인건비**: 파견한 용병 수 × 티어별 단가
- 비용은 파견 시작 시 선차감됨 (실패해도 돌려받지 못함)

### 골드 획득 경로

| 경로 | 금액 |
|------|------|
| 의뢰 성공 | 유형 × 난이도 배수 |
| 여행 중 골드 발견 | +20~120G |
| 방치형 보상 | 분당 1G, 최대 480G |
| 이동 선택지 이벤트 | safe [0~25G] / risky [25~60G] / hidden [120G+] |

### 골드 소비 경로

| 경로 | 금액 |
|------|------|
| 용병 유료 모집 | 가변 |
| 의뢰 파견비용 | 난이도/시간 비례 |
| 인건비 | 용병 수 × 티어 단가 |
| 시설 건설 | `baseCost × multiplier^(lv-1)` (기하급수적 증가) |
| 용병 방출 퇴직금 | 인건비 × 레벨 |
| 트레잇 삭제 | acquired 200G / evolved 500G |

## 야영지 시설

야영지에 시설을 건설하고 업그레이드하면 용병단 전체의 능력이 향상됩니다. 시설 건설에는 **골드와 실제 시간**이 소요되며(OGame 스타일), 한 번에 하나의 시설만 건설할 수 있습니다. 취소 시 골드는 전액 환불됩니다.

**총 12종 시설, 최대 레벨 25.**

### 기반 시설

| 시설 | 효과 | 분위기 |
|------|------|--------|
| 훈련소 | 퀘스트 경험치 보너스 (최대 +100%) | 용병들의 훈련 시설 |
| 의무실 | 부상 회복 시간 단축 (최대 -70%) + 트레잇 삭제 해금 | 부상 용병 치료 |
| 주둔지 | 용병 최대 인원 증가 (최대 +20명) | 더 많은 용병 수용 |
| 정보망 | 의뢰 목록 추가 생성 (최대 +5개) | 지역 정보 네트워크 |

### 전문 시설

| 시설 | 효과 | 분위기 |
|------|------|--------|
| 주점 | 고급 용병 모집 확률 상승 (최대 +15%) | 용병 소문이 모이는 곳 |
| 방어시설 | 여행 이벤트 피해 감소 (최대 -50%) | 야영지 방어 강화 |
| 금고 | 방치 보상 상한 증가 (최대 +480G) | 골드 보관 및 이자 수익 |
| 이동수단 | 이동 시간 단축 (최대 -40%) | 말, 마차 등 이동 수단 |
| 야전병원 | 부상 발생 확률 감소 (최대 -50%) | 전장 의료 지원 |

### 미래 시설 (stub)

| 시설 | 예정 효과 | 상태 |
|------|----------|------|
| 대장간 | 장비 강화 효과 | 장비 시스템 구현 후 활성화 |
| 연구소 | 특수 트레잇 해금 | 향후 구현 예정 |
| 게시판 | 특수 의뢰 알림 | 향후 구현 예정 |

**건설 시간 예시 (훈련소 기준):** Lv1 = 5분, Lv2 = 10분, Lv5 ≈ 1시간, Lv10 ≈ 12시간, Lv20 ≈ 30일, Lv25 ≈ 4개월

## 여행 이벤트

이동 중에 다양한 이벤트가 발생할 수 있습니다. 이동 거리가 길수록 이벤트 발생 확률이 높아집니다(최대 80%).

| 이벤트 유형 | 예시 |
|-----------|------|
| 발견 | 버려진 보물 발견, 골드 획득 |
| 도적 습격 | 골드 손실, 용병 피해 |
| 날씨 이변 | 이동 지연 |
| 행운 | 예상치 못한 이득 |
| 조우 | 이동 중 특수 상황 |

(총 12종 이벤트)

## 이동 선택지 이벤트 (M3)

이동이 완료된 뒤 확률적으로 **"회상" 팝업**이 등장합니다. 방치형 UX를 유지하면서도 여행 중 벌어진 일을 선택지 형식으로 풀어냅니다.

- **12종 시나리오**: 조우(encounter), 딜레마(dilemma), 발견(discovery), 위험(hazard) 4개 카테고리
- **선택지 구조**: 안전한 선택(safe) + 위험한 선택(risky) + 숨겨진 선택(hidden, 특정 트레잇 보유 시 표시)
- **효과 8종**: 골드 획득/손실, 경험치, 명성, 부상, 피로 회복, 아이템, 선천 트레잇 부여
- **기대값 정책**: safe ≤ risky < hidden. 숨겨진 선택지의 기대값이 가장 높아 탐험 보상으로 기능

발동 확률은 이동 거리와 리전 티어에 따라 달라지며, 상한 30%로 제한된다. 파견 화면에서 전원이 파견 중일 때는 발동하지 않는다.

## 지역 조사

현재 위치한 리전을 더 깊이 탐색할 수 있습니다. 용병 1명을 조사에 배치하면 실제 시간이 흐른 뒤 결과를 받습니다.

**조사 특징:**
- 퀘스트 파견·이동과 **완전히 독립된 별도 슬롯** — 조사 중에도 퀘스트 파견 가능
- 단, 이동과는 상호 배제 (이동 중 조사 불가, 조사 중 이동 불가)
- 소요 시간: 리전 등급에 따라 Tier 1 = 5분, Tier 5 = 20분

**지식 포인트와 발견:**

조사가 성공할 때마다 리전의 **지식 포인트**(0~100)가 증가합니다. 지식이 특정 임계값에 도달하면 숨겨진 정보가 자동으로 드러납니다.

| 트리거 유형 | 조건 | 결과 |
|-----------|------|------|
| 세력 단서 발견 | 세력별 고정 임계값 | 비밀/지역 세력 단계적 공개 |
| 숨겨진 연계 퀘스트 활성화 | 60~95 (체인별 상이) | 체인 퀘스트 1단계 파견 화면에 등장 |
| 지역 변형 | 80(유적지) / 90(마을) / 95(숨겨진) | 섹터가 영구 변형, 전용 퀘스트 풀 생성 |

**성공률:** 파견 용병의 AGI와 VIT에 따라 결정됩니다.

---

## 연계 퀘스트 (체인 퀘스트, M3)

지역 조사를 통해 발견되는 **다단계 서사 퀘스트**입니다. 지식 임계값 도달 시 1단계가 파견 화면에 자동으로 등장하며, 단계를 완료할수록 이야기가 이어집니다.

- **7개 체인 / 24단계**: 리전 Tier 1~5에 걸쳐 분포. 일부 체인은 이동을 요구
- **파견 화면 최상단 고정**: 진행 중인 체인 단계는 일반 퀘스트보다 항상 위에 표시
- **주인공 용병 시스템**: 1단계 최초 성공 시 기여도 1위 용병이 "주인공"으로 고정. 이후 단계에서도 같은 용병이 서사에 등장. 사망 시 폴백 재지정
- **체인 단계 사망률 50% 감소**: 체인 퀘스트 파견 시 주인공 용병의 사망 위험이 일반 퀘스트 대비 절반
- **완주 보상**: 체인을 끝까지 완료하면 명성 보너스(단계 수 × 150 × 티어 가중치)와 확정 아이템 보상 지급
- **14일 휴면**: 14일 이상 진행이 없으면 체인이 휴면(dormant) 상태로 전환. 파견 화면에서 재활성화 가능
- **재발동 불가**: 완주한 체인은 다시 시작되지 않음

단계 간에는 `next_step_delay`(다음 단계 출현까지 실제 대기 시간)가 적용된다.

## 지역 변형 (M3)

지역 조사를 통해 지식 포인트가 최고 임계값에 도달하면 해당 리전의 섹터 하나가 **영구적으로 변형**됩니다. 변형은 되돌릴 수 없으며, 이동 화면에서 특수 아이콘으로 구분됩니다.

### 변형 유형

| 유형 | 아이콘 | 특성 | 전용 퀘스트 |
|------|--------|------|-----------|
| 마을(village) | 🏘️ | 안전·반복 수익형 | 호위/탐험/노동 위주, 난이도 D2~D3 |
| 유적지(ruins) | 🏛️ | 고위험·고보상, 일부 드랍 플래그 | 토벌/탐험/약탈 위주, 난이도 D4~D5 |
| 숨겨진 섹터(hidden) | ✨ | 희귀 아이템·트레잇 학습 가속 | 특수 플래그 퀘스트 위주 |

변형된 섹터에는 해당 유형에 맞는 전용 퀘스트가 자동으로 생성됩니다. 일부 퀘스트에는 완료 시 특수 효과(트레잇 학습 가속, 길드 장비 드랍 등)가 붙습니다.

현재 18개 리전이 변형 트리거 대상이며, 리전당 최대 1개 섹터가 변형됩니다.

---

## 퀘스트 서사 (M3)

모든 퀘스트 결과에 **서사 텍스트**가 붙습니다. 같은 퀘스트 유형이더라도 결과(대성공/성공/실패/대실패)마다 다른 문체와 분위기의 텍스트가 등장합니다.

- **88개 템플릿**: 퀘스트 유형(6종) × 결과(4종) 매트릭스. 유형·결과별 4개 변형
- **TemplateEngine 적용**: `{merc.name}`, `{quest.enemy}`, `{region.name}` 등 변수 치환으로 자연스러운 문장 생성
- **엘리트 전용 서사**: 엘리트 몬스터가 등장하는 퀘스트는 별도 8개 템플릿 사용
- **대표 용병**: 파티 내 전투력 기여가 가장 높은 용병이 서사의 주인공으로 선정됨
- 완료 팝업에 이탤릭 서사 영역으로 표시되며, 활동 로그에도 서사 첫 줄이 기록됨

## 세력 시스템

대륙에는 14개의 **세력**(조직·집단)이 존재합니다. 세력과의 관계를 쌓으면 다양한 혜택을 얻을 수 있습니다.

### 세력 유형

| 유형 | 수 | 설명 |
|------|-----|------|
| 공개 | 6 | 처음부터 이름과 정보가 공개됨 |
| 비밀 | 4 | 지역 조사를 통해 단서를 모아야 실체가 드러남 |
| 지역 | 4 | 특정 지역에 뿌리를 둔 세력. 조사로 발견 가능 |

### 세력 발견

비밀·지역 세력은 **세력 단서**를 모아 단계적으로 정보가 공개됩니다.

| 단계 | 공개 정보 | 활동 로그 |
|------|----------|----------|
| 단서 1 | 세력의 존재 암시 (이름 비공개) | "세력 단서 발견" |
| 단서 2 | 세력 이름·설명 공개 | "세력 발견: {이름}의 정체를 파악했다" |
| 단서 3 | 거점 위치 공개 | "거점 발견: {이름}의 전초기지 위치를 파악했다" |

### 세력 가입

세력에 가입하면 해당 세력의 패시브 혜택을 받을 수 있습니다. 동시에 최대 **3개 세력**에 가입할 수 있습니다.

**가입 조건 (세력마다 다름):**
- 세력 평판 > 0 (필수)
- 세력 단서 3단계 완료 (일부 비밀 세력)
- 용병단 랭크 조건 충족 (일부 세력)
- 충돌 세력과의 가입 수 제한

**평판 시스템:**
- 미가입 상태: 최대 10까지 누적
- 가입 후: 최대 100까지 누적 / 최소 -100

### 충돌 관계

일부 세력들은 서로 대립합니다. 충돌하는 세력에 가입하면:
- 기존에 가입한 충돌 세력에서 **자동 탈퇴**
- 해당 세력의 **평판 -20** 패널티

### 패시브 혜택

각 세력은 고유한 혜택을 제공한다. 세력에 가입하면 퀘스트 보상 배수, 성공률, 회복 시간, 모집 비용 등 다양한 수치에 실제 영향을 미친다. 명성 랭크 보너스와 세력 패시브는 `PassiveBonusService`를 통해 함께 누적되어 적용된다.

## 활동 로그

플레이어의 모든 주요 행동은 활동 로그에 기록됩니다. 최대 100개의 로그가 유지됩니다.

기록되는 사건:
- 의뢰 완료 결과 (성공/실패/대성공/대실패) + 서사 텍스트
- 용병 상태 변화 (부상, 사망, 회복)
- 이동 완료
- 용병 모집
- 용병 방출
- 용병 레벨업
- 시설 건설 완료
- 지역 조사 완료/실패
- 세력 단서 발견 / 세력 실체 파악 / 거점 발견
- 연계 퀘스트 단계 진행 / 완주
- 지역 변형 발생
- 이동 선택지 이벤트 결과
